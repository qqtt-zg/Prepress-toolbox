进度条至少显示300ms，确保用户能看到：
```csharp
if (displayTime < MIN_PROGRESS_DISPLAY_TIME_MS)

    var remainingTime = (int)(MIN_PROGRESS_DISPLAY_TIME_MS - displayTime);

```

### 3. UI强制刷新
使用 `Refresh()` 确保进度条立即显示：

所有操作的catch块都调用 `HideBottomProgress()`：
```csharp

}
```
