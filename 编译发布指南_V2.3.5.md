## 编译步骤

### 步骤 1：清理项目

```powershell
# 在项目根目录执行
cd src/WindowsFormsApp3
dotnet clean
```

或在 Visual Studio 中：
- 菜单：生成 → 清理解决方案

### 步骤 2：还原 NuGet 包

```powershell
# 在项目根目录执行
cd src/WindowsFormsApp3
dotnet restore
```

或在 Visual Studio 中：
- 右键点击解决方案 → 还原 NuGet 包

### 步骤 3：编译项目（Release 配置）

#### 方法 1：使用 Visual Studio

1. 打开 `WindowsFormsApp3.sln`
2. 切换到 Release 配置：
   - 工具栏：Debug → Release
3. 选择平台：
   - 工具栏：Any CPU → x64
4. 编译：
   - 菜单：生成 → 生成解决方案
   - 或按 `Ctrl+Shift+B`

#### 方法 2：使用命令行

```powershell
# 在项目根目录执行
cd src/WindowsFormsApp3
dotnet build -c Release -r win-x64
```

### 步骤 4：验证编译结果

检查编译输出目录：
```
src/WindowsFormsApp3/bin/Release/net48/win-x64/
```

确认以下文件存在：
- `大诚重命名工具.exe`
- `大诚重命名工具.exe.config`
- 所有依赖的 DLL 文件
- `x64/pdfium.dll`（PdfiumViewer 依赖）
- `Resources/dc.ico`（应用程序图标）

## 创建安装包

### 步骤 1：打开 Inno Setup Compiler

1. 启动 Inno Setup Compiler
2. 打开脚本文件：
   - 文件 → 打开
   - 选择 `installers/Setup.iss`

### 步骤 2：验证脚本配置

检查以下关键配置：

```ini
[Setup]
AppVersion=2.3.5
AppVerName=大诚重命名工具 v2.3.5
OutputBaseFilename=大诚重命名工具_v2.3.5_安装包
```

### 步骤 3：编译安装包

#### 方法 1：使用 Inno Setup Compiler GUI

1. 在 Inno Setup Compiler 中：
   - 菜单：生成 → 编译
   - 或按 `Ctrl+F9`

2. 等待编译完成
3. 查看编译日志，确认无错误

#### 方法 2：使用命令行

```powershell
# 在项目根目录执行
cd installers
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" Setup.iss
```

### 步骤 4：验证安装包

检查输出目录：
```
installers/安装包/
```

确认文件存在：
- `大诚重命名工具_v2.3.5_安装包.exe`

检查文件大小：
- 应该在 50-100 MB 之间（取决于依赖项）

## 常见问题

### Q1：编译失败，提示缺少 NuGet 包？

**解决方法**：
```powershell
cd src/WindowsFormsApp3
dotnet restore
```

### Q2：Inno Setup 找不到源文件？

**解决方法**：
1. 检查 Setup.iss 中的路径是否正确
2. 确认编译输出目录存在：`src/WindowsFormsApp3/bin/Release/net48/win-x64/`
3. 重新编译项目

### Q3：安装包运行时提示缺少 .NET Framework 4.8？

**解决方法**：
- 安装包会自动检测并提示下载
- 或手动下载：https://dotnet.microsoft.com/download/dotnet-framework/net48

### Q4：版本号显示不正确？

**解决方法**：
1. 检查 `AssemblyInfo.cs` 中的版本号
2. 重新编译项目（Clean + Rebuild）
3. 重新创建安装包

### Q5：pdfium.dll 缺失？

**解决方法**：
1. 确认 `x64/pdfium.dll` 存在于编译输出目录
2. 检查 Setup.iss 中的文件复制配置：
   ```ini
   Source: "..\src\WindowsFormsApp3\bin\Release\net48\win-x64\x64\pdfium.dll"; DestDir: "{app}\x64"
   ```
