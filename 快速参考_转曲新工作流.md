# 快速参考 - 转曲新工作流

## 🎯 核心变化

### 之前的工作流
```
点击"转曲" → 选择保存位置 → 保存文件 → 完成
```

### 现在的工作流
```
点击"转曲" → 预览效果 → 点击"保存"或"另存为" → 完成
```

## 📋 操作步骤

### 快速转曲并保存
1. 打开PDF
2. 点击"检查器"
3. 点击"转曲"
4. 点击"保存"（覆盖原文件）

### 转曲并另存为
1. 打开PDF
2. 点击"检查器"
3. 点击"转曲"
4. 点击"另存为"（保存到新文件）

### 转曲后撤回（新增！）
1. 打开PDF
2. 点击"检查器"
3. 点击"转曲"
4. 预览效果不满意
5. 点击"撤回"（恢复原始PDF）

## 🔘 按钮说明

### 工具栏按钮
| 按钮 | 功能 | 启用条件 |
|------|------|----------|
| 打开PDF | 打开PDF文件 | 始终启用 |
| 保存 | 覆盖当前文件 | 有未保存更改时 |
| 另存为 | 保存到新文件 | 加载PDF后 |
| 撤回 | 撤回转曲操作 | 有未保存更改时 |
| 检查器 | 打开检查器窗口 | 加载PDF后 |

### 检查器按钮
| 按钮 | 功能 | 位置 |
|------|------|------|
| 转曲 | 将文字转换为路径 | 字体标签页工具栏 |

## 💡 提示信息

### 转曲完成后
```
✅ 字体已转换为路径，请使用工具栏的保存或另存为按钮保存文件
```

### 状态栏显示
- 加载文件: `已加载: 文件名.pdf`
- 转曲完成: `已转曲 (未保存) - 文件名.pdf`
- 保存完成: `已保存: 文件名.pdf`

## ⚠️ 注意事项

1. **转曲后必须手动保存**: 转曲操作不会自动保存文件
2. **可以撤回**: 保存前可以随时点击"撤回"按钮恢复原始PDF
3. **预览是临时的**: 关闭应用程序前请保存，否则转曲结果会丢失
4. **保存会覆盖**: "保存"按钮会覆盖原文件，请谨慎使用
5. **另存为更安全**: 建议使用"另存为"保存到新文件
6. **撤回不可逆**: 保存后无法撤回，请在保存前确认效果

## 🔍 常见问题

### Q: 转曲后看不到效果？
A: 转曲完成后预览会自动刷新，如果没有刷新请检查日志

### Q: 保存按钮是灰色的？
A: 保存按钮只在有未保存更改时启用，转曲后会自动启用

### Q: 转曲后关闭了应用程序，文件没有保存？
A: 转曲结果存储在内存中，关闭前必须手动保存

### Q: 如何保留原文件？
A: 使用"另存为"按钮保存到新文件，原文件不会被修改

### Q: 转曲后发现效果不满意怎么办？
A: 点击"撤回"按钮即可恢复到原始PDF，然后可以重新调整参数再次转曲

### Q: 保存后还能撤回吗？
A: 不能，保存后转曲数据已写入文件，无法撤回。请在保存前确认效果
