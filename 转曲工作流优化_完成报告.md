```csharp
public byte[] ConvertTextToOutlinesBytes(string inputPath)
{
    string tempOutput = Path.Combine(Path.GetTempPath(), $"temp_outline_{Guid.NewGuid()}.pdf");
    bool success = ConvertTextToOutlines(inputPath, tempOutput);
    if (success && File.Exists(tempOutput))
    {
        byte[] result = File.ReadAllBytes(tempOutput);
        File.Delete(tempOutput);
        return result;
    }
    return null;
}
```

## 新增功能

### 1. 转曲不立即保存
- 点击"转曲"按钮后，只在内存中执行转曲
- 不弹出保存对话框
- 显示提示："字体已转换为路径，请使用工具栏的保存或另存为按钮保存文件"

### 2. 另存为按钮
- 位置：工具栏"保存"按钮之后
- 功能：保存到新文件（不覆盖原文件）
- 默认文件名：原文件名 + "_转曲.pdf"
- 保存后询问是否打开新文件

### 3. 完善的保存按钮
- 功能：保存到当前文件（覆盖原文件）
- 状态：只在有未保存更改时启用
- 保存后自动重新加载文件

### 4. 状态管理
- 跟踪是否有未保存的更改
- 状态栏显示当前状态
- 按钮状态根据文件状态动态更新

### 5. 预览功能
- 转曲完成后自动预览转曲效果
- 使用临时文件实现预览
- 不影响原文件

## 事件流程

```
用户点击"转曲"按钮
    ↓
PdfInspectorControl.OutlineButton_Click
    ↓
调用 PdfFontOutlineService.ConvertTextToOutlinesBytes
    ↓
触发 FontOutlineCompleted 事件
    ↓
PdfInspectorForm 转发事件
    ↓
PdfOperationsPanel.Inspector_FontOutlineCompleted
    ↓
保存字节数组、启用按钮、创建临时文件、重新加载预览
    ↓
用户点击"保存"或"另存为"
    ↓
写入文件
```

## 用户使用流程

### 方式1: 转曲并保存（覆盖原文件）
1. 打开PDF文件
2. 点击"检查器"按钮
3. 在"字体"标签页点击"转曲"按钮
4. 确认转曲操作
5. 等待转曲完成（自动预览）
6. 点击工具栏的"保存"按钮
7. 完成

### 方式2: 转曲并另存为（保存到新文件）
1. 打开PDF文件
2. 点击"检查器"按钮
3. 在"字体"标签页点击"转曲"按钮
4. 确认转曲操作
5. 等待转曲完成（自动预览）
6. 点击工具栏的"另存为"按钮
7. 选择保存位置和文件名
8. 选择是否打开新文件
9. 完成
