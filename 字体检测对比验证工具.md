## 自动化验证脚本

创建 `test_font_detection.bat`：

```batch
@echo off
setlocal enabledelayedexpansion

echo ========================================
echo 字体检测对比验证工具
echo ========================================
echo.

set TEST_PDF=C:\Users\admin\Desktop\45 HYS000714 60张 HED-125 B270喇叭形吊灯（玫瑰金色）273876 铭牌.pdf

echo [1] 测试 pdffonts 命令行工具
echo ----------------------------------------
if exist "poppler\bin\pdffonts.exe" (
    echo ✓ pdffonts.exe 存在
    echo.
    echo 执行: pdffonts.exe "%TEST_PDF%"
    echo.
    poppler\bin\pdffonts.exe "%TEST_PDF%"
    echo.
) else (
    echo ✗ pdffonts.exe 不存在
    echo 请确保 poppler\bin\pdffonts.exe 已安装
    pause
    exit /b 1
)

echo.
echo [2] 检查编译输出目录
echo ----------------------------------------
set BIN_DIR=src\WindowsFormsApp3\bin\Debug\net48\win-x64

if exist "%BIN_DIR%\poppler\bin\pdffonts.exe" (
    echo ✓ 编译输出目录包含 pdffonts.exe
) else (
    echo ✗ 编译输出目录缺少 pdffonts.exe
    echo 正在复制...
    xcopy /E /I /Y poppler "%BIN_DIR%\poppler"
    if !errorlevel! equ 0 (
        echo ✓ 复制成功
    ) else (
        echo ✗ 复制失败
    )
)

echo.
echo [3] 查看最新日志
echo ----------------------------------------
if exist "app_2026-01-20.log" (
    echo 最近的字体检测日志:
    findstr /C:"[Poppler]" app_2026-01-20.log | findstr /C:"解析字体" | more
) else (
    echo ✗ 日志文件不存在
)

echo.
echo ========================================
echo 测试完成
echo ========================================
echo.
echo 下一步:
echo 1. 运行程序测试字体检测功能
echo 2. 对比上述输出与 Adobe Acrobat 结果
echo 3. 检查日志中的 "嵌入: SubsetEmbedded" 是否正确
echo.
pause
```

## 常见问题诊断

### 问题1: 字体数量为0

**症状**：
```
[Poppler] 解析完成，共 0 个字体
```

**可能原因**：
1. PDF文件已转曲（所有文字转为路径）
2. 解析逻辑错误
3. pdffonts输出格式不符合预期

**诊断步骤**：
```cmd
# 1. 手动运行pdffonts查看输出
poppler\bin\pdffonts.exe "测试文件.pdf"

# 2. 检查是否有输出
# 3. 如果有输出但程序解析为0，说明解析逻辑有问题
```

### 问题2: 嵌入状态错误

**症状**：
```
[Poppler] ✓ 解析字体: MicrosoftYaHei (类型: CID Type 0C, 嵌入: NotEmbedded)
```
但pdffonts显示 `yes yes yes`

**原因**：列位置解析错误（已在新代码中修复）

**验证修复**：
查看日志中的 `解析结果` 行：
```
[Poppler] 解析结果: name='...', type='...', encoding='...', emb='yes', sub='yes', uni='yes'
```

如果 `emb='yes'` 但最终状态是 `NotEmbedded`，说明判断逻辑有问题。

### 问题3: 字体名称包含子集前缀

**症状**：
显示 `EPROCH+MicrosoftYaHei` 而不是 `MicrosoftYaHei`

**原因**：子集前缀移除逻辑未执行

**检查代码**：
```csharp
// 应该在最后执行
if (Regex.IsMatch(fontInfo.FontName, @"^[A-Z]{6}\+"))
{
    fontInfo.FontName = fontInfo.FontName.Substring(7);
}
```
