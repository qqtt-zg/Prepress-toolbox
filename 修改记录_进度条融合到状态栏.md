# 修改记录：进度条融合到状态栏

## 修改原因

用户反馈：进度条没有和文件名结合在一起，共用一个区域。

**原设计问题**:
- JavaScript进度条在HTML层（`viewer.html`）
- C#状态栏在WinForms层（`PdfOperationsPanel`）
- 两者是独立的层，无法真正融合

**新设计**:
- 将进度条移到C#状态栏中
- 进度条作为状态栏的背景Panel
- 文字Label在进度条上方，始终可见

### 2. PdfOperationsPanel.cs 修改

#### 添加进度条控制方法

```csharp
/// <summary>
/// 显示底部进度条
/// </summary>
private void ShowBottomProgress()
{
    if (InvokeRequired)
    {
        Invoke(new Action(ShowBottomProgress));
        return;
    }
    
    _progressBar.Visible = true;
    _progressBar.Width = 0;
}

/// <summary>
/// 更新底部进度条
/// </summary>
private void UpdateBottomProgress(int percent)
{
    if (InvokeRequired)
    {
        Invoke(new Action<int>(UpdateBottomProgress), percent);
        return;
    }
    
    int targetWidth = (int)((statusPanel.Width - 20) * percent / 100.0);
    _progressBar.Width = targetWidth;
}

/// <summary>
/// 隐藏底部进度条
/// </summary>
private void HideBottomProgress()
{
    if (InvokeRequired)
    {
        Invoke(new Action(HideBottomProgress));
        return;
    }
    
    _progressBar.Visible = false;
    _progressBar.Width = 0;
}
```

## 颜色配置

### 进度条颜色
```csharp
// ARGB格式
Color.FromArgb(
    20,   // Alpha: 20 (半透明，约8%不透明度)
    24,   // Red
    144,  // Green
    255   // Blue
)
```

**效果**: 半透明蓝色 `rgba(24, 144, 255, 0.08)`

### 深色主题适配
可以在 `ApplyTheme` 方法中动态调整：
```csharp
public void ApplyTheme(bool isDark)
{
    if (isDark)
    {
        // 深色主题：更明显的进度条
        _progressBar.BackColor = Color.FromArgb(38, 24, 144, 255); // 15%不透明度
    }
    else
    {
        // 浅色主题：更淡的进度条
        _progressBar.BackColor = Color.FromArgb(20, 24, 144, 255); // 8%不透明度
    }
}
```

## 布局结构

```
statusPanel (Panel)
├── _progressBar (Panel) - Dock.Left, Width动态变化
│   └── BackColor: 半透明蓝色
└── _lblStatus (Label) - Dock.Left, BackColor.Transparent
    └── Text: "已加载: xxx.pdf"
```

**关键点**:
1. `_progressBar` 先添加（作为背景）
2. `_lblStatus` 后添加（在上层）
3. `_lblStatus.BackColor = Transparent` 让进度条可见
4. 两者都是 `Dock.Left`，自动排列
