## 1. 产品概述

### 1.1 什么是大诚工具箱？

大诚工具箱是一款专为**印刷行业印前处理**设计的 Windows 桌面应用程序。它将印前工作中最常见、最繁琐的操作自动化，帮助您高效完成以下任务：

| 核心功能 | 说明 |
|---------|------|
| **文件批量重命名** | 根据正则匹配、Excel 数据自动重命名 PDF 文件，支持材料/数量/尺寸/工艺等参数组合 |
| **Excel 数据驱动** | 导入 Excel 表格，自动匹配订单号、数量、材料等信息到文件 |
| **PDF 拆分** | 按页数范围将一个 PDF 拆分为多个独立文件 |
| **骑马订拼版** | 自动计算骑马订页面排列，生成拼版 PDF |
| **PDF 预览与管理** | 多标签页 PDF 预览、字体检查、页面框检查 |
| **综合设置** | 正则管理、材料管理、主题定制等全方位配置 |

### 1.2 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64位） |
| 运行环境 | .NET Framework 4.8（安装包自动包含） |
| 磁盘空间 | 至少 500 MB 可用空间 |
| 可选组件 | Ghostscript（字体转曲功能需要，详见 [FAQ](#13-常见问题解答)） |

### 1.3 首次启动

双击 `大诚工具箱.exe` 启动程序。首次启动时，系统会自动完成以下初始化：

1. 检查数据完整性并自动修复（如有异常）
2. 加载配置文件
3. 检查更新（需要网络连接）
4. 打开主界面，默认进入"首页"面板

> 💡 **提示**: 程序采用单实例运行模式。如果已经有一个实例在运行，再次双击图标会将已运行的窗口激活到前台，而不会启动新实例。

## 3. 基本操作流程 — 文件批量重命名

文件批量重命名是大诚工具箱的**核心功能**，也是启动后的默认页面。本节介绍最基本的使用流程。

### 3.1 界面总览

文件重命名面板分为三个区域：

```
┌──────────────────────────────────────────────────────────┐
│  顶部控制栏（两行）                                        │
│  第一行: [输入目录▼] [选择文件夹] [正则▼] [监控] [悬浮拖拽]  │
│  第二行: [JSON▼] [保存] [导入表格] [应用数据] [清除] [导出]   │
│          [复制/剪切] [手动模式] [批量模式] [开始更改]         │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  文件列表表格（15列）                                      │
│  序号│原文件名│新文件名│正则结果│订单号│材料│数量│尺寸│...   │
│                                                          │
├──────────────────────────────────────────────────────────┤
│  状态栏: 操作状态 │ 模式状态 │ 组合预览                     │
└──────────────────────────────────────────────────────────┘
```

### 3.2 快速上手：3 步完成重命名

#### 步骤 1：选择输入目录

点击 **"选择文件夹"** 按钮，选择包含待处理 PDF 文件的文件夹。

> 也可以直接在输入目录下拉框中输入路径，或从历史记录中选择。

#### 步骤 2：选择正则规则

在 **正则表达式** 下拉框中选择一个匹配规则，用于从文件名中提取关键字（如订单号）。

常用正则规则：
| 规则名 | 表达式 | 匹配示例 |
|--------|--------|---------|
| 订单号(4位数字) | `(\d{4})` | 文件名 `1234_标签.pdf` → 提取 `1234` |
| 订单号(5位数字) | `(\d{5})` | 文件名 `12345_海报.pdf` → 提取 `12345` |
| 订单号(6位数字) | `(\d{6})` | 文件名 `123456_包装.pdf` → 提取 `123456` |

#### 步骤 3：启动监控并处理文件

1. 点击 **"开始监控"** 按钮，开始监控输入目录
2. 将 PDF 文件放入监控目录（或通过悬浮拖拽窗口导入）
3. 根据当前模式处理文件：
   - **手动模式**：自动弹出材料选择对话框 → 选择参数 → 立即执行重命名
   - **批量模式**：文件自动添加到表格 → 检查数据 → 点击"开始更改"统一执行

### 3.3 操作模式详解

#### 手动模式（适合逐个确认）

```
放入文件 → 弹出材料选择对话框 → 选择材料/数量/尺寸等 → 确认 → 立即重命名
```

**适用场景**: 每个文件的参数不同，需要人工确认；文件数量较少。

**操作步骤**:
1. 确认 **"手动模式"** 按钮已高亮（默认启用）
2. 点击 **"开始监控"**
3. 将 PDF 文件放入监控目录
4. 系统检测到新文件后，自动弹出 **材料选择对话框**
5. 在对话框中选择材料、输入数量、确认尺寸等参数
6. 点击"确定"，文件立即被重命名并复制/移动到导出目录

#### 批量模式（适合大批量处理）

```
放入多个文件 → 自动添加到表格 → 自动匹配Excel数据 → 检查 → 点击"开始更改"统一执行
```

**适用场景**: 文件数量多，参数可通过 Excel 数据自动匹配；需要先预览再统一执行。

**操作步骤**:
1. 点击 **"批量模式"** 按钮（"手动模式"按钮自动取消高亮）
2. 先导入 Excel 数据（详见[第4章](#4-excel-数据导入与匹配)）
3. 点击 **"开始监控"**
4. 将 PDF 文件放入监控目录
5. 文件自动添加到表格，Excel 数据自动匹配
6. 匹配成功的行背景变为**浅绿色**
7. 检查表格数据无误后，点击 **"开始更改"**
8. 系统批量执行重命名操作

#### 复制模式 vs 剪切模式

| 模式 | 行为 | 原文件 |
|------|------|--------|
| **复制模式**（默认） | 复制文件到导出目录并重命名 | ✅ 保留 |
| **剪切模式** | 移动文件到导出目录并重命名 | ❌ 删除 |

> ⚠️ **注意**: 剪切模式下原文件会被移走，请确保操作正确后再使用。

### 3.4 文件列表表格

表格共 15 列，展示每个文件的完整信息：

| 列名 | 说明 | 数据来源 |
|------|------|---------|
| 序号 | 自动递增编号 | 自动生成 |
| 原文件名 | 源文件名 | 自动读取 |
| 新文件名 | 重命名后的文件名 | 自动组合生成 |
| 正则结果 | 从文件名提取的关键字 | 正则匹配 |
| 订单号 | 订单编号 | Excel匹配 / 手动输入 |
| 材料 | 材料类型（PET、PP等） | Excel匹配 / 材料对话框 |
| 数量 | 印刷数量 | Excel匹配 / 材料对话框 |
| 尺寸 | 成品尺寸（宽×高+出血） | PDF自动读取 / 手动输入 |
| 形状 | 形状代号 | 材料对话框 |
| 列组合 | Excel多列组合值 | Excel匹配 |
| 平张 | 平张排版布局数 | 自动计算 |
| 卷装 | 卷装排版布局数 | 自动计算 |
| 工艺 | 颜色模式+膜类型 | 材料对话框 |
| 页数 | PDF页数 | 自动读取 |
| 时间 | 处理时间 | 自动记录 |

#### 右键菜单快捷操作

在表格单元格上右键，根据当前列显示不同快捷菜单：

| 右键位置 | 可用操作 |
|---------|---------|
| **材料列** | 快捷选择材料（PET、PP、PVC等） |
| **数量列** | 批量设置数量 / 批量增减数量 |
| **尺寸列** | 设置出血值 |
| **订单号列** | 批量生成订单号（可设前缀、起始号、增量、位数、后缀） |
| **原文件名列** | 提取数量（从文件名中按单位字符提取数字） |
| **任意列** | 复制、删除行、刷新 |

> 💡 **技巧**: 右键列头可以隐藏不需要的列、保存列配置或恢复原始布局。

### 3.5 新文件名生成规则

新文件名由多个字段按特定顺序组合而成，组合规则由 **事件分组配置**（设置 → 事件分组）决定。

默认组合格式示例：
```
正则结果_订单号_材料_数量_尺寸_形状_工艺.pdf
```

例如：`1234_ORD001_PET_500_210×297+3_R_彩色光膜.pdf`

状态栏右侧的 **"组合"** 预览会实时显示当前配置的效果。

## 4. Excel 数据导入与匹配

Excel 数据导入是批量模式的核心支撑，通过 Excel 表格中的数据自动填充文件参数。

### 4.1 导入 Excel 数据

1. 点击 **"导入表格"** 按钮，弹出 Excel 导入对话框
2. 点击 **"选择文件"**，选择 Excel 文件（支持 `.xlsx`、`.xls` 格式）
3. 配置以下参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| **搜索列** | Excel 中用于匹配的列 | 选择"订单号"列 |
| **返回列** | 匹配成功后提取数据的列 | 选择"数量"列 |
| **序号列** | 可选，提取序号的列 | 选择"序号"列 |
| **正则表达式** | 从文件名提取匹配关键字的正则 | `(\d{4})` 提取4位订单号 |
| **列组合** | 可选，选择多列并用分隔符组合 | 选择"客户+备注"列 |

4. 点击 **"确定"** 完成导入

### 4.2 数据匹配流程

导入 Excel 后，系统自动执行匹配：

```
文件名 → 正则提取关键字 → 在Excel搜索列中查找匹配 → 提取返回列/序号列数据 → 填入表格
```

**匹配规则**:
- 使用**精确匹配**：文件名中提取的关键字必须与 Excel 搜索列的值完全一致
- **一对多匹配**（文件分裂）：如果一个关键字匹配到多行 Excel 数据，会为每行创建一个文件副本
- 匹配成功的行背景变为**浅绿色**

### 4.3 手动触发匹配

如果需要重新匹配（如修改了正则规则），点击 **"应用数据"** 按钮即可重新执行匹配。

### 4.4 Excel 格式建议

为确保匹配成功，Excel 表格应满足以下条件：

| 要求 | 说明 |
|------|------|
| 首行为表头 | 系统自动跳过首行 |
| 搜索列无重复值 | 如有重复，取第一个匹配项 |
| 数据格式统一 | 订单号等字段格式保持一致 |
| 无合并单元格 | 合并单元格可能导致读取异常 |

**示例 Excel 表格**:

| 订单号 | 客户名 | 数量 | 材料 | 备注 |
|--------|--------|------|------|------|
| 1001 | 张三 | 500 | PET | 加急 |
| 1002 | 李四 | 1000 | PP | 常规 |
| 1003 | 王五 | 2000 | PVC | 特殊工艺 |

- 显示 PDF 的原始尺寸（自动读取）
- 可手动调整宽度和高度
- 出血值可在设置中预设（默认 3mm）

#### 形状选择

| 选项 | 代号 | 说明 |
|------|------|------|

- **颜色模式**: 彩色 / 黑白
- **膜类型**: 无膜 / 光膜 / 哑膜 / 其他

- **平张模式**: 适用于单张纸印刷
- **卷装模式**: 适用于卷筒材料印刷
- **一式N联**: 设置联数和份数，用于多联排版

#### 预设功能

右键对话框空白处可保存当前参数为预设，下次使用时可直接加载，避免重复输入。

## 6. 悬浮拖拽窗口

悬浮拖拽窗口是一个始终置顶的小窗口，方便您从任何位置拖入文件。

### 6.1 启用悬浮窗口

点击顶部控制栏的 **"悬浮拖拽"** 按钮，弹出悬浮窗口。

### 6.2 使用方法

1. 从资源管理器中选择 PDF 文件
2. 拖拽到悬浮窗口上
3. 文件自动复制/移动到监控目录
4. 系统检测到新文件后，按当前模式处理

### 6.3 拖拽操作模式

## 8. 拼版工作台 — 骑马订

拼版工作台集成了 PDF 预览和拼版功能，目前实现了**骑马订拼版**。

### 8.1 骑马订原理

骑马订是将书页对折后从折缝处钉合的装订方式。拼版时需要将页面按特定顺序排列在大纸上，使得折叠后页码顺序正确。

以 16 页为例（4 张纸，每张正反两面）：

| 纸张 | 正面左侧 | 正面右侧 | 反面左侧 | 反面右侧 |
|------|----------|----------|----------|----------|
| 第1张 | 第1页 | 第16页 | 第2页 | 第15页 |
| 第2张 | 第3页 | 第14页 | 第4页 | 第13页 |
| 第3张 | 第5页 | 第12页 | 第6页 | 第11页 |
| 第4张 | 第7页 | 第10页 | 第8页 | 第9页 |

> ⚠️ 总页数必须是 4 的倍数，不足时系统自动补空白页。

### 8.2 界面布局

```

│  左侧面板                    │  右侧预览区                │

│                              │                            │
│  ┌─ 文件列表 ──────┐        │                            │
│  │ 文件名│页数│状态│        │     PDF 预览内容            │
│  │       │    │    │        │                            │
│  └────────────────┘        │                            │
│                              │                            │
│  功能选择:                    │                            │
│  [骑马订] [N-Up] [裁切] [重排]│                            │
│                              │                            │
│  纸张大小: [A4 ▼]            │                            │
```

### 8.3 操作步骤

#### 步骤 1：加载 PDF 文件

三种方式：
- 点击 **"加载 PDF 文件"** 按钮，在文件对话框中选择（支持多选）
- 将 PDF 文件直接 **拖拽** 到文件列表区域
- 将包含 PDF 的 **文件夹** 拖拽到文件列表区域

#### 步骤 2：选择骑马订功能

点击左侧 **"骑马订"** 按钮（默认已选中，高亮显示）。

#### 步骤 3：配置参数

| 参数 | 选项 | 默认值 | 说明 |
|------|------|--------|------|
| **纸张大小** | A3 / A4 / A5 / B4 / SRA3 | A4 | 成品尺寸（对折后的大小） |
| **爬移补偿** | 浮点数（mm） | 0.0 | 内层纸张向内的偏移补偿 |

**纸张尺寸对应表**:

| 选项 | 成品尺寸 (mm) | 大纸尺寸 (mm) |
|------|---------------|---------------|
| A3 | 297 × 420 | 594 × 420 |
| A4 | 210 × 297 | 420 × 297 |
| A5 | 148 × 210 | 296 × 210 |
| B4 | 257 × 364 | 514 × 364 |
| SRA3 | 320 × 450 | 640 × 450 |

**关于爬移补偿**:
- 骑马订装订时，内层纸张折缝处会向内偏移
- 补偿值越大，内层偏移越多
- 一般薄纸小册子可设为 0，厚纸册子建议设 0.5-2.0mm

#### 步骤 4：生成 PDF

1. 点击 **"生成 PDF"** 按钮
2. 系统自动：分析页数 → 补空白页 → 计算排列 → 生成拼版 PDF
3. 输出文件自动保存到源 PDF 同目录，命名格式：`{原文件名}_saddle_stitch_{时间戳}.pdf`
4. 生成完成后自动在预览区打开

#### 步骤 5：管理文件

- **切换文件**: 在文件列表中点击不同行，右侧预览自动切换
- **删除文件**: 选中文件后按 `Delete` 键移除

## 13. 常见问题解答

### Q1: 启动时提示"Ghostscript 未安装"

**原因**: 字体转曲功能依赖 Ghostscript。  
**解决**: 
1. 下载安装 Ghostscript（参考安装指南）
2. 安装后重启大诚工具箱
3. 如果不使用字体转曲功能，可以忽略此提示

### Q2: 导入 Excel 后匹配不上数据

**可能原因及解决方法**:

| 原因 | 解决方法 |
|------|---------|
| 正则表达式不匹配文件名 | 检查正则规则是否正确，使用"测试正则"功能验证 |
| 搜索列数据格式不一致 | 确保 Excel 中的数据格式与文件名提取的关键字完全一致 |
| Excel 有合并单元格 | 取消合并单元格后重新导入 |
| 首行不是表头 | 确保首行为表头，系统会自动跳过 |

### Q3: 文件重命名后找不到输出文件

**解决方法**:
1. 检查 **设置 → 导出路径** 中的输出目录配置
2. 检查文件操作模式（复制/剪切），剪切模式下原文件会被移走
3. 在材料选择对话框中确认导出路径设置

### Q4: PDF 预览显示异常或卡顿

**可能原因及解决方法**:

| 原因 | 解决方法 |
|------|---------|
| PDF 文件过大 | 尝试压缩 PDF 后再预览 |
| CefSharp 初始化未完成 | 等待几秒后重试 |
| 显卡驱动问题 | 更新显卡驱动 |

### Q5: 监控目录没有自动检测到新文件

**可能原因及解决方法**:

| 原因 | 解决方法 |
|------|---------|
| 未点击"开始监控" | 确认监控按钮显示为"停止监控"状态 |
| 文件放入了错误的目录 | 检查输入目录路径是否正确 |
| 文件正在被其他程序占用 | 关闭占用文件的程序后重试 |
| 网络驱动器监控延迟 | 网络路径的文件监控可能有延迟，建议使用本地路径 |

### Q6: 骑马订拼版后页码顺序不对

**解决方法**:
1. 确认源 PDF 的页码顺序是正确的（第1页在前）
2. 检查总页数是否为 4 的倍数（系统会自动补空白页）
3. 确认纸张大小选择正确

### Q7: 程序崩溃后数据丢失

**解决方法**:
1. 开启 **设置 → 常规 → 当日JSON自动创建/加载**
2. 设置合理的自动保存频率（建议 60 秒）
3. 程序异常退出后，下次启动会提示恢复自动保存的数据
4. 也可以手动点击"保存"按钮保存当前配置

### Q8: 如何修改文件名的组合格式？

**解决方法**:
1. 进入 **设置 → 事件分组** 标签页
2. 将不需要的字段禁用（取消勾选）
3. 拖拽调整字段的排列顺序
4. 修改分组前缀（如将 `&ID-` 改为 `ORD-`）
5. 保存设置后，首页状态栏的"组合"预览会实时更新

### Q9: 如何添加自定义正则表达式？

**解决方法**:
1. 进入 **设置 → 正则管理** 标签页
2. 在"名称"框输入规则名称（如"客户编号"）
3. 在"正则表达式"框输入表达式（如 `([A-Z]{2}\d{3})`）
4. 点击"添加"
5. 使用"测试"功能验证表达式是否正确

### Q10: 窗口找不到了怎么办？

**解决方法**:
1. 检查系统托盘（右下角），双击大诚工具箱图标恢复窗口
2. 如果设置了全局快捷键，按快捷键切换窗口显示
3. 如果以上方法无效，在任务管理器中结束进程后重新启动

## 14. 操作技巧与注意事项

### 14.1 效率提升技巧

| 技巧 | 说明 |
|------|------|
| 🔥 **使用悬浮拖拽窗口** | 无需切换到程序窗口，从任何位置拖入文件即可处理 |
| 🔥 **善用预设功能** | 在材料选择对话框中保存常用参数预设，避免重复输入 |
| 🔥 **批量模式 + Excel** | 大批量文件处理时，先用 Excel 准备好数据，再使用批量模式自动匹配 |
| 🔥 **JSON 配置保存** | 每次工作结束后保存 JSON 配置，下次可快速恢复工作状态 |
| 🔥 **右键快捷操作** | 在表格列上右键可快速设置材料、数量、订单号等 |
| 🔥 **自定义快捷键** | 在设置中配置全局热键，快速切换窗口显隐 |

### 14.2 数据安全注意事项

| 注意事项 | 说明 |
|---------|------|
| ⚠️ **剪切模式不可逆** | 使用剪切模式前请确认操作正确，原文件会被移走 |
| ⚠️ **备份重要文件** | 建议在批量操作前备份原始文件 |
| ⚠️ **检查匹配结果** | 批量模式下执行前，务必检查表格中的匹配结果是否正确 |
| ⚠️ **开启自动保存** | 建议开启当日 JSON 自动保存，防止意外崩溃导致数据丢失 |
| ⚠️ **导出路径检查** | 定期检查导出路径是否有效，避免文件输出到错误位置 |

### 14.3 性能优化建议

| 建议 | 说明 |
|------|------|
| 🚀 **使用本地路径** | 监控目录和导出路径尽量使用本地磁盘，网络路径会有延迟 |
| 🚀 **控制文件数量** | 单次处理的文件数量建议不超过 500 个，过多可能导致界面卡顿 |
| 🚀 **关闭不需要的列** | 在表格列头右键隐藏不需要的列，减少渲染开销 |
| 🚀 **合理设置自动保存** | 自动保存频率不宜过低（建议不低于 30 秒），频繁保存会影响性能 |

### 14.4 键盘快捷操作

| 快捷键 | 功能 |
|--------|------|
| `Delete` | 删除选中的文件/作业 |
| `Ctrl+C` | 复制选中单元格内容 |
| 全局热键 | 切换窗口显示/隐藏（在设置中配置） |
