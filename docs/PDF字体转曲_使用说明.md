# PDF 字体转曲功能 - 使用说明

## 功能概述

PDF 检查器现已集成 Ghostscript 实现的字体转曲功能，可以将 PDF 中的文字转换为矢量路径。

---

## 前置要求

### 安装 Ghostscript

字体转曲功能需要 Ghostscript 支持。请按以下步骤安装：

#### Windows 系统

1. **下载 Ghostscript**
   - 访问官网：[https://ghostscript.com/releases/gsdnld.html](https://ghostscript.com/releases/gsdnld.html)
   - 下载最新版本（推荐 10.x 或更高版本）
   - 选择对应版本：
     - 64位系统：`gs10.04.0-win64.exe`
     - 32位系统：`gs10.04.0-win32.exe`

2. **安装 Ghostscript**
   - 运行安装程序
   - 使用默认安装路径（推荐）
   - 完成安装

3. **验证安装**
   - 打开命令提示符（CMD）
   - 输入：`gswin64c --version`（64位）或 `gswin32c --version`（32位）
   - 如果显示版本号，说明安装成功

#### 便携式部署（无需安装）

如果不想在系统中安装 Ghostscript，可以将其打包到应用程序中：

1. 下载 Ghostscript 安装包
2. 解压或安装到临时目录
3. 复制以下文件到应用程序目录：
   - `gswin64c.exe`（或 `gswin32c.exe`）
   - `gsdll64.dll`（或 `gsdll32.dll`）
   - `lib` 文件夹（包含 Ghostscript 库文件）

应用程序会自动检测并使用本地的 Ghostscript。

## 使用方法

### 1. 打开 PDF 检查器

1. 在 PDF 操作面板加载 PDF 文件
2. 点击"检查器"按钮
3. 检查器窗口将打开

### 2. 查看字体信息

1. 点击"字体"标签页
2. 查看 PDF 中使用的所有字体
3. 检查字体嵌入状态和问题

### 3. 执行字体转曲

1. 点击"转曲"按钮（带闪电图标）
2. 阅读确认对话框中的警告信息
3. 点击"确定"继续
4. 在保存对话框中选择输出位置
   - 默认文件名：`原文件名_转曲.pdf`
   - 默认位置：原文件所在目录
5. 点击"保存"开始转曲
6. 等待处理完成
7. 选择是否在检查器中打开转曲后的文件

## 功能说明

### 转曲参数

应用程序使用以下 Ghostscript 参数：

```bash
-o output.pdf              # 输出文件
-dNoOutputFonts            # 将字体转换为路径
-sDEVICE=pdfwrite          # 使用 PDF 写入设备
-dPDFSETTINGS=/prepress    # 高质量输出（适合印刷）
-dCompatibilityLevel=1.4   # PDF 1.4 兼容性
input.pdf                  # 输入文件
```

### 转曲效果

**转曲前**：
- ✅ 文字可以选择和复制
- ✅ 文字可以搜索
- ✅ 文件较小
- ❌ 依赖字体文件
- ❌ 可能出现字体替换问题

**转曲后**：
- ✅ 不依赖字体文件
- ✅ 显示完全一致
- ✅ 适合印刷和交付
- ❌ 文字无法选择和复制
- ❌ 文字无法搜索
- ❌ 文件可能变大

## 故障排除

### 问题 1：提示"未找到 Ghostscript"

**原因**：系统中未安装 Ghostscript，或应用程序无法找到它。

**解决方案**：
1. 检查 Ghostscript 是否已安装
2. 确认安装路径是否正确
3. 尝试将 Ghostscript 可执行文件复制到应用程序目录
4. 查看日志文件了解详细信息

### 问题 2：转曲失败

**原因**：PDF 文件损坏、受保护或格式不兼容。

**解决方案**：
1. 检查 PDF 文件是否可以正常打开
2. 确认 PDF 文件没有密码保护
3. 尝试使用其他 PDF 工具修复文件
4. 查看日志文件了解详细错误信息

### 问题 3：转曲后文件很大

**原因**：文字转换为路径后，文件大小会增加，尤其是包含大量中文字体的文件。

**解决方案**：
1. 这是正常现象，无需担心
2. 如果文件过大，可以考虑：
   - 只转曲必要的字体
   - 使用 PDF 压缩工具
   - 降低图像质量（如果包含图像）

### 问题 4：转曲后图像质量下降

**原因**：Ghostscript 默认会重新处理图像。

**解决方案**：
1. 当前版本使用 `-dPDFSETTINGS=/prepress` 参数，已经是最高质量
2. 如果仍有问题，可以联系开发者调整参数

## 常见问题

### Q: 转曲需要多长时间？
**A**: 取决于 PDF 的页数、文字数量和复杂度，通常几秒到几十秒。

### Q: 转曲后可以撤销吗？
**A**: 不可以。转曲是不可逆操作，请保留原始文件。

### Q: 转曲会修改原文件吗？
**A**: 不会。转曲操作会创建新文件，原文件保持不变。

### Q: 可以只转曲部分字体吗？
**A**: 当前版本会转曲所有字体。未来版本可能支持选择性转曲。

### Q: 转曲后 PDF 版本会改变吗？
**A**: 会。转曲后的 PDF 版本为 1.4（兼容性最好）。

### Q: Ghostscript 是免费的吗？
**A**: Ghostscript 有两种许可证：
- **AGPL**：免费使用，但要求应用程序也开源
- **商业许可**：需要购买，可以闭源使用

对于个人和开源项目，可以免费使用 AGPL 版本。
