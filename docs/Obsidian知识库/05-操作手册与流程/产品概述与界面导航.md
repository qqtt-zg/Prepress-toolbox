## 什么是大诚工具箱？

大诚工具箱是一款专为**印刷行业印前处理**设计的 Windows 桌面应用程序。

| 核心功能 | 说明 |
|---------|------|
| **文件批量重命名** | 根据正则匹配、Excel 数据自动重命名 PDF 文件 |
| **Excel 数据驱动** | 导入 Excel 表格，自动匹配订单号、数量、材料等信息 |
| **PDF 拆分** | 按页数范围将一个 PDF 拆分为多个独立文件 |
| **骑马订拼版** | 自动计算骑马订页面排列，生成拼版 PDF |
| **PDF 预览与管理** | 多标签页 PDF 预览、字体检查、页面框检查 |
| **综合设置** | 正则管理、材料管理、主题定制等全方位配置 |

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64位） |
| 运行环境 | .NET Framework 4.8（安装包自动包含） |
| 磁盘空间 | 至少 500 MB 可用空间 |
| 可选组件 | Ghostscript（字体转曲功能需要） |

## 首次启动

双击 `大诚工具箱.exe` 启动程序。首次启动时系统会自动完成初始化：检查数据完整性、加载配置、检查更新、打开主界面。

> 程序采用单实例运行模式。再次双击图标会将已运行的窗口激活到前台。

## 导航按钮说明

| 按钮 | 功能 | 快速说明 |
|------|------|---------|
| 📁 **首页** | 文件批量重命名 | 核心功能面板，支持正则匹配、Excel驱动、自动监控 |
| 🗄 **数据库** | Excel 数据查看 | 以表格形式展示已导入的 Excel 数据 |
| 🖥 **工作台** | AE 风格工作台 | 多 PDF 文件管理与预览 |
| 📄 **PDF拆分** | PDF 按页拆分 | 根据 Excel 数据按页数范围拆分 PDF |
| ⚙ **设置** | 综合设置 | 8 个分类设置页 |
| ··· **菜单** | 更多选项 | 打开日志 / 关于 / 退出 |

## 侧边栏折叠

- 点击左下角 **"折叠"** 按钮，侧边栏收缩为仅显示图标的窄栏（70px）
- 再次点击 **"展开"** 按钮，恢复完整导航栏（140px）

## 窗口控制

| 操作 | 方法 |
|------|------|
| 最小化 | 点击标题栏 **─** 按钮 |
| 最大化/还原 | 点击标题栏 **□** 按钮 |
| 关闭 | 点击标题栏 **✕** 按钮 → 最小化到系统托盘（不退出） |
| 完全退出 | 右键系统托盘图标 → 退出 |
| 全局热键 | 按自定义快捷键切换窗口显示/隐藏 |

> 关闭窗口后程序仍在系统托盘运行，双击托盘图标可恢复窗口。
