# PDF 拆分操作

> 来源：`docs/用户操作教程.md` §7

PDF 拆分功能可根据 Excel/CSV 数据表中定义的文件名和页数，将一个源 PDF 按页数范围自动拆分为多个独立文件。

## 界面说明

```
┌──────────────────────────────────────────────────────────┐
│  📄 源PDF: [________________________] [选择文件]  页数: 32 │
│  📊 Excel: [________________________] [选择文件]          │
├──────────────────────────────────────────────────────────┤
│  序号 │ 文件名 │ 页数 │ 起始页 │ 结束页                   │
│   1   │ 封面   │  4   │   1    │   4                     │
│   2   │ 内页A  │  8   │   5    │   12                    │
│   3   │ 内页B  │  12  │   13   │   24                    │
│   4   │ 封底   │  4   │   25   │   28                    │
├──────────────────────────────────────────────────────────┤
│  [执行拆分]  [取消]  ████████░░░░  状态: 就绪             │
└──────────────────────────────────────────────────────────┘
```

## 操作步骤

### 步骤 1：导入源 PDF

- **拖拽**：将 `.pdf` 文件拖拽到源 PDF 输入框
- **浏览**：点击 **"选择文件"** 按钮

导入后自动显示源 PDF 总页数。

### 步骤 2：导入 Excel/CSV 数据

- **拖拽**：将 `.xlsx`、`.xls` 或 `.csv` 文件拖拽到 Excel 输入框
- **浏览**：点击 **"选择文件"** 按钮
- **双击** Excel 路径框可直接打开该文件查看/编辑

**Excel 格式要求**：

| 列 | 内容 | 是否必须 | 说明 |
|----|------|---------|------|
| 第1列 | 文件名 | 必须 | 输出 PDF 的文件名 |
| 第2列 | 页数 | 必须 | 该输出文件包含的页数 |
| 第3列 | 订单号 | 可选 | 仅用于记录 |
| 第4列 | 数量 | 可选 | 仅用于记录 |

> 首行视为表头，自动跳过。

### 步骤 3：查看预览

PDF 和 Excel 都导入后，系统自动计算页数范围并在预览表格中显示。

- ✅ 正常：显示 `就绪 - 将拆分为 N 个文件`
- ⚠️ 页数超出：显示红色警告

### 步骤 4：执行拆分

1. 点击 **"执行拆分"**
2. 选择输出目录
3. 确认操作
4. 等待进度条完成

**输出规则**：
- 输出文件名自动补 `.pdf` 后缀
- 同名文件已存在时自动添加序号后缀
- 超出源 PDF 页数范围的条目会被跳过

> 拆分过程中可随时点击 **"取消"** 中断操作。
