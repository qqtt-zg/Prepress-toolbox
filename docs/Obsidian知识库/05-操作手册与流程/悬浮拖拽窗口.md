# 悬浮拖拽窗口

> 来源：`docs/用户操作教程.md` §6

悬浮拖拽窗口是一个始终置顶的小窗口，方便从任何位置拖入文件。

## 启用悬浮窗口

点击顶部控制栏的 **"悬浮拖拽"** 按钮，弹出悬浮窗口。

## 使用方法

1. 从资源管理器中选择 PDF 文件
2. 拖拽到悬浮窗口上
3. 文件自动复制/移动到监控目录
4. 系统检测到新文件后，按当前模式处理

## 拖拽操作模式

通过顶部控制栏的下拉框选择：

| 模式 | 行为 |
|------|------|
| **默认（智能判断）** | 同分区自动移动，跨分区自动复制 |
| **强制复制** | 始终复制文件到监控目录 |
| **强制移动** | 始终移动文件到监控目录 |

## 窗口特性

- 无边框设计，可拖动移动位置
- 支持边缘拖拽调整大小
- 窗口位置和大小自动记忆
- 右键菜单：显示边框 / 隐藏窗口
- 支持 Popcat 图标模式（在主题设置中开启）
