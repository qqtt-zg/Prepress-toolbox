# 常见问题解答

> 来源：`docs/用户操作教程.md` §13

## Q1: 启动时提示"Ghostscript 未安装"

**原因**：字体转曲功能依赖 Ghostscript。

**解决**：
1. 下载安装 Ghostscript（参考安装指南）
2. 安装后重启大诚工具箱
3. 如果不使用字体转曲功能，可以忽略此提示

## Q2: 导入 Excel 后匹配不上数据

| 原因 | 解决方法 |
|------|---------|
| 正则表达式不匹配文件名 | 检查正则规则，使用"测试正则"功能验证 |
| 搜索列数据格式不一致 | 确保 Excel 数据格式与提取的关键字完全一致 |
| Excel 有合并单元格 | 取消合并单元格后重新导入 |
| 首行不是表头 | 确保首行为表头 |

## Q3: 文件重命名后找不到输出文件

1. 检查 **设置 → 导出路径** 中的输出目录配置
2. 检查文件操作模式（复制/剪切），剪切模式下原文件会被移走
3. 在材料选择对话框中确认导出路径设置

## Q4: PDF 预览显示异常或卡顿

| 原因 | 解决方法 |
|------|---------|
| PDF 文件过大 | 尝试压缩 PDF 后再预览 |
| CefSharp 初始化未完成 | 等待几秒后重试 |
| 显卡驱动问题 | 更新显卡驱动 |

## Q5: 监控目录没有自动检测到新文件

| 原因 | 解决方法 |
|------|---------|
| 未点击"开始监控" | 确认监控按钮显示为"停止监控"状态 |
| 文件放入了错误的目录 | 检查输入目录路径是否正确 |
| 文件正在被其他程序占用 | 关闭占用文件的程序后重试 |
| 网络驱动器监控延迟 | 建议使用本地路径 |

## Q6: 骑马订拼版后页码顺序不对

1. 确认源 PDF 的页码顺序是正确的
2. 检查总页数是否为 4 的倍数（系统会自动补空白页）
3. 确认纸张大小选择正确

## Q7: 程序崩溃后数据丢失

1. 开启 **设置 → 常规 → 当日JSON自动创建/加载**
2. 设置合理的自动保存频率（建议 60 秒）
3. 程序异常退出后，下次启动会提示恢复自动保存的数据
4. 也可以手动点击"保存"按钮保存当前配置

## Q8: 如何修改文件名的组合格式？

1. 进入 **设置 → 事件分组** 标签页
2. 将不需要的字段禁用
3. 拖拽调整字段的排列顺序
4. 修改分组前缀
5. 保存设置后，状态栏"组合"预览会实时更新

## Q9: 如何添加自定义正则表达式？

1. 进入 **设置 → 正则管理** 标签页
2. 输入规则名称和表达式
3. 点击"添加"
4. 使用"测试"功能验证

## Q10: 窗口找不到了怎么办？

1. 检查系统托盘（右下角），双击图标恢复窗口
2. 如果设置了全局快捷键，按快捷键切换窗口显示
3. 如果以上方法无效，在任务管理器中结束进程后重新启动
