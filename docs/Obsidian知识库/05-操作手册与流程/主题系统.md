# 主题系统与外观定制

> 来源：`docs/用户操作教程.md` §12

## 内置主题

| 主题名 | 特点 |
|--------|------|
| **深色**（默认） | 深灰背景，低饱和度强调色，护眼 |
| **浅色** | 白色背景，蓝色强调，清爽明亮 |
| **护眼绿** | 绿色系背景，绿色强调，长时间使用舒适 |
| **经典蓝** | 蓝色系背景，蓝色强调，专业感 |

## 切换主题

1. 进入 **设置 → 主题** 标签页
2. 从下拉框中选择主题
3. 点击 **"应用"** 按钮，界面立即更新

## 自定义主题

可自定义以下颜色组：

| 颜色组 | 包含项 |
|--------|--------|
| 背景色 | 主背景、卡片背景、面板背景、输入框背景 |
| 文字色 | 主文字、次要文字 |
| 边框色 | 统一边框 |
| 强调色 | 主要（蓝）、成功（绿）、警告（黄）、错误（红） |
| 特殊强调 | 行数（蓝）、列数（绿）、数量（橙）、旋转（紫） |
| 交互色 | 激活背景、悬停背景 |
| 滚动条 | 深色/浅色模式 |

## 主题操作

| 操作 | 方法 |
|------|------|
| 新建主题 | 点击"新建"，基于当前主题克隆 |
| 保存主题 | 点击"保存" |
| 应用主题 | 点击"应用"，立即更新界面 |
| 重置主题 | 点击"重置"（仅内置主题可用） |
| 删除主题 | 点击"删除"（仅自定义主题可删除） |
| 导出主题 | 点击"导出"，保存为 JSON 文件 |
| 导入主题 | 点击"导入"，从 JSON 文件加载 |

## 悬浮拖拽窗口外观

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 默认宽度 | 180px | 悬浮窗口宽度 |
| 默认高度 | 80px | 悬浮窗口高度 |
| 背景色 | #4682B4 | 悬浮窗口背景色 |
| 拖拽时背景色 | #326496 | 拖拽文件时的背景色 |
| 透明度 | 92% | 悬浮窗口透明度 |
| Popcat图标模式 | 关闭 | 启用后使用透明背景+图标 |
