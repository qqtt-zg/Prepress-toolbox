# 材料选择对话框

> 来源：`docs/用户操作教程.md` §5

材料选择对话框是手动模式下的核心交互界面，每个新文件都会弹出此对话框供确认参数。

## 对话框布局

```
┌─────────────────────────────────────────────────────────┐
│  材料选择                                                │
├─────────────────────────────────────────────────────────┤
│  材料选择:  [PET] [PP] [PVC] [纸] [不干胶] ... [更多▼]   │
│                                                          │
│  数量:    [________]     序号: [___]     订单号: [______] │
│                                                          │
│  尺寸:    宽[___] × 高[___]  +出血[___]                  │
│                                                          │
│  形状:    ○直角  ○圆形  ○异形  ○圆角矩形                  │
│                                                          │
│  工艺:    颜色: [彩色▼]  膜: [光膜▼]                      │
│                                                          │
│  导出路径: [________________________] [浏览]              │
│                                                          │
│  ┌─ 排版设置 ──────────────────────────────────┐         │
│  │ 模式: ○平张  ○卷装    一式N联: [__]联 [__]份  │         │
│  │ 标识页: ☐ 添加标识页                          │         │
│  └──────────────────────────────────────────────┘         │
│                              [取消]  [确定]              │
└─────────────────────────────────────────────────────────┘
```

## 材料选择

提供 15 个常用材料快捷按钮：

| 按钮 | 材料 | 按钮 | 材料 |
|------|------|------|------|
| PET | PET 聚酯 | PP | PP 聚丙烯 |
| PVC | PVC 聚氯乙烯 | 纸 | 纸张类 |
| 不干胶 | 不干胶贴纸 | 铝箔 | 铝箔材料 |
| 牛皮纸 | 牛皮纸 | 铜版纸 | 铜版纸 |
| 白卡纸 | 白卡纸 | 瓦楞纸 | 瓦楞纸 |
| 透明PVC | 透明PVC | 珠光纸 | 珠光纸 |
| 特种纸 | 特种纸 | 更多▼ | 下拉扩展选项 |

## 数量输入

- 支持单个数量（如 `500`）
- 支持逗号分隔多个数量（如 `500,1000,2000`），系统为每个数量生成一个文件副本

## 尺寸调整

- 显示 PDF 原始尺寸（自动读取）
- 可手动调整宽度和高度
- 出血值可在设置中预设（默认 3mm）

## 形状选择

| 选项 | 代号 | 说明 |
|------|------|------|
| 直角 | Z | 无圆角的矩形 |
| 圆形 | C | 正圆 |
| 异形 | Y | 不规则形状 |
| 圆角矩形 | R+数字 | 数字表示圆角半径 |

## 工艺选择

- **颜色模式**：彩色 / 黑白
- **膜类型**：无膜 / 光膜 / 哑膜 / 其他

## 排版设置

- **平张模式**：适用于单张纸印刷
- **卷装模式**：适用于卷筒材料印刷
- **一式N联**：设置联数和份数，用于多联排版

## 预设功能

右键对话框空白处可保存当前参数为预设，下次使用时直接加载，避免重复输入。
