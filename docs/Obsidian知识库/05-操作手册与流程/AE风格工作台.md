# AE 风格工作台

> 来源：`docs/用户操作教程.md` §9

AE 工作台采用类似 Adobe After Effects 的界面布局，集成了 PDF 预览、作业管理和日志功能。

## 界面布局

```
┌──────────────────────────────────────────────────────────┐
│  工具栏: [添加PDF] [移除] [运行] [停止]                    │
├──────────┬───────────────────────────────────────────────┤
│          │  [预览] [详情] [日志]                           │
│  作业    │                                               │
│   ├ 队列 │         标签页内容区                           │
│   ├ 完成 │                                               │
│   └ 失败 │                                               │
│  工具    ├───────────────────────────────────────────────┤
│   ├ 拼版 │  作业列表                                      │
│   └ 印前 │  文件名 │ 页数 │ 状态                          │
├──────────┴───────────────────────────────────────────────┤
│  状态栏: 就绪                                             │
└──────────────────────────────────────────────────────────┘
```

## 操作步骤

1. **添加 PDF**：点击 **"添加PDF"** 按钮，或拖拽文件到作业列表
2. **浏览预览**：在作业列表中点击某行，右侧"预览"标签页自动显示该 PDF
3. **查看日志**：切换到 **"日志"** 标签页查看操作记录
4. **移除作业**：选中作业后点击 **"移除"** 按钮或按 `Delete` 键

> **注意**："运行"和"停止"按钮目前尚未接入执行逻辑，仅输出日志消息。
