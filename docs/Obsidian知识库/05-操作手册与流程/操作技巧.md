# 操作技巧与注意事项

> 来源：`docs/用户操作教程.md` §14

## 效率提升技巧

| 技巧 | 说明 |
|------|------|
| **使用悬浮拖拽窗口** | 无需切换到程序窗口，从任何位置拖入文件即可处理 |
| **善用预设功能** | 在材料选择对话框中保存常用参数预设，避免重复输入 |
| **批量模式 + Excel** | 大批量文件处理时，先用 Excel 准备好数据，再使用批量模式自动匹配 |
| **JSON 配置保存** | 每次工作结束后保存 JSON 配置，下次可快速恢复工作状态 |
| **右键快捷操作** | 在表格列上右键可快速设置材料、数量、订单号等 |
| **自定义快捷键** | 在设置中配置全局热键，快速切换窗口显隐 |

## 数据安全注意事项

| 注意事项 | 说明 |
|---------|------|
| **剪切模式不可逆** | 使用剪切模式前请确认操作正确，原文件会被移走 |
| **备份重要文件** | 建议在批量操作前备份原始文件 |
| **检查匹配结果** | 批量模式下执行前，务必检查表格中的匹配结果是否正确 |
| **开启自动保存** | 建议开启当日 JSON 自动保存，防止意外崩溃导致数据丢失 |
| **导出路径检查** | 定期检查导出路径是否有效，避免文件输出到错误位置 |

## 性能优化建议

| 建议 | 说明 |
|------|------|
| **使用本地路径** | 监控目录和导出路径尽量使用本地磁盘，网络路径会有延迟 |
| **控制文件数量** | 单次处理的文件数量建议不超过 500 个 |
| **关闭不需要的列** | 在表格列头右键隐藏不需要的列，减少渲染开销 |
| **合理设置自动保存** | 自动保存频率不宜过低（建议不低于 30 秒） |

## 键盘快捷操作

| 快捷键 | 功能 |
|--------|------|
| `Delete` | 删除选中的文件/作业 |
| `Ctrl+C` | 复制选中单元格内容 |
| 全局热键 | 切换窗口显示/隐藏（在设置中配置） |
