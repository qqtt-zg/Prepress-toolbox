# 综合设置详解

> 来源：`docs/用户操作教程.md` §11

点击左侧导航栏的 **"设置"** 按钮进入综合设置面板，包含 8 个标签页。

## 常规设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 分隔符 | `_` | 文件名各字段之间的分隔符 |
| 单位 | `mm` | 尺寸显示单位 |
| 窗口透明度 | 100% | 主窗口透明度 |
| 快捷键 | 无 | 最小化/恢复窗口的全局快捷键 |
| 重命名完成通知 | 关闭 | 文件重命名完成后是否弹出通知 |
| 当日JSON自动创建/加载 | 开启 | 自动创建/加载当日工作 JSON 文件 |
| 自动保存频率 | 60秒 | 自动保存间隔 |
| 同时输出排版模式布局数 | 关闭 | 是否同时输出两种布局模式的数量 |
| 尺寸显示大数在前 | 开启 | 尺寸显示时较大数值放在前面 |
| 一式N联自动填入数量 | 开启 | 一式N联排版时自动填入数量 |

## 正则管理

管理用于文件名匹配的正则表达式规则。

| 操作 | 方法 |
|------|------|
| 添加正则 | 输入名称和表达式，点击"添加" |
| 删除正则 | 选中列表中的行，点击"删除" |
| 测试正则 | 输入测试文本，选中正则后查看匹配结果 |
| 编辑正则 | 选中行，内容自动回填，修改后重新添加 |

> 正则表达式使用第一个捕获组 `()` 的值作为匹配结果。

## 材料形状

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 默认材料 | 空 | 文件命名时使用的默认材料名称 |
| 出血值 | 3,5,10 | 预设出血值列表 |
| 零形代码 | Z | 直角形状的标识代码 |
| 圆角代码 | R | 圆角形状的标识代码 |
| 椭圆代码 | Y | 椭圆形状的标识代码 |
| 圆形代码 | C | 圆形形状的标识代码 |
| 隐藏半径值 | 关闭 | 是否隐藏半径/圆角数值 |

## 导出路径

| 操作 | 方法 |
|------|------|
| 添加路径 | 点击"添加"，选择文件夹 |
| 删除路径 | 选中行后点击"删除" |
| 调整优先级 | 使用"上移"/"下移"按钮 |
| 包含子文件夹 | 勾选对应行的复选框 |

> 不存在的路径会显示"(路径不存在)"后缀。

## 自动排版

**平张模式**：纸张宽度/高度、上/下/左/右边距

**卷筒模式**：固定宽度、最小长度、上/下/左/右边距

**通用设置**：行数、列数

> 提供"从PDF获取尺寸"按钮，可自动读取 PDF 首页尺寸并填入。

## 事件分组

管理文件名组合时各字段的分组和排列顺序。

**9 个默认分组**：

| 分组 | 前缀 | 默认排序 |
|------|------|---------|
| 订单组 | &ID- | 1 |
| 材料组 | &MT- | 2 |
| 数量组 | &DN- | 3 |
| 工艺组 | &DP- | 4 |
| 客户组 | &CU- | 5 |
| 备注组 | &MK- | 6 |
| 行数组 | &Row- | 7 |
| 列数组 | &Col- | 8 |
| 未分组 | 无 | 9 |

**11 个可配置事件项**：正则结果、订单号、材料、数量、工艺、尺寸、序号、列组合、材料类型、行数、列数

**预设系统**：内置"全功能配置"预设，可创建、保存、加载、删除自定义预设。

## 字体文字

**文字项管理**：8 个可选项，支持启用/禁用、拖拽排序、实时预览。

**字体管理**：选择 PDF 兼容字体，中日韩字体优先显示，实时预览字体效果。

## 保存设置

- 每个标签页的修改可独立保存
- 点击底部 **"保存所有设置"** 统一保存
- 保存后立即生效，无需重启
