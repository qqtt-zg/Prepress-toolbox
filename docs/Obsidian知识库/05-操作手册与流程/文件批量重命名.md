# 文件批量重命名

> 来源：`docs/用户操作教程.md` §3

文件批量重命名是大诚工具箱的**核心功能**，也是启动后的默认页面。

## 界面总览

文件重命名面板分为三个区域：

```
┌──────────────────────────────────────────────────────────┐
│  顶部控制栏（两行）                                        │
│  第一行: [输入目录▼] [选择文件夹] [正则▼] [监控] [悬浮拖拽]  │
│  第二行: [JSON▼] [保存] [导入表格] [应用数据] [清除] [导出]   │
│          [复制/剪切] [手动模式] [批量模式] [开始更改]         │
├──────────────────────────────────────────────────────────┤
│  文件列表表格（15列）                                      │
│  序号│原文件名│新文件名│正则结果│订单号│材料│数量│尺寸│...   │
├──────────────────────────────────────────────────────────┤
│  状态栏: 操作状态 │ 模式状态 │ 组合预览                     │
└──────────────────────────────────────────────────────────┘
```

## 快速上手：3 步完成重命名

### 步骤 1：选择输入目录

点击 **"选择文件夹"** 按钮，选择包含待处理 PDF 文件的文件夹。也可在下拉框中输入路径或从历史记录选择。

### 步骤 2：选择正则规则

在 **正则表达式** 下拉框中选择匹配规则，用于从文件名中提取关键字。

| 规则名 | 表达式 | 匹配示例 |
|--------|--------|---------|
| 订单号(4位数字) | `(\d{4})` | `1234_标签.pdf` → `1234` |
| 订单号(5位数字) | `(\d{5})` | `12345_海报.pdf` → `12345` |
| 订单号(6位数字) | `(\d{6})` | `123456_包装.pdf` → `123456` |

### 步骤 3：启动监控并处理文件

1. 点击 **"开始监控"**
2. 将 PDF 文件放入监控目录
3. 按当前模式处理：手动模式弹出材料选择对话框；批量模式自动添加到表格

## 操作模式

### 手动模式（逐个确认）

```
放入文件 → 弹出材料选择对话框 → 选择参数 → 确认 → 立即重命名
```

适用场景：每个文件参数不同，需要人工确认。

### 批量模式（大批量处理）

```
放入多个文件 → 自动添加到表格 → 自动匹配Excel → 检查 → 点击"开始更改"
```

适用场景：文件数量多，参数可通过 Excel 自动匹配。

### 复制模式 vs 剪切模式

| 模式 | 行为 | 原文件 |
|------|------|--------|
| **复制模式**（默认） | 复制文件到导出目录并重命名 | 保留 |
| **剪切模式** | 移动文件到导出目录并重命名 | 删除 |

> 剪切模式下原文件会被移走，请确保操作正确后再使用。

## 文件列表表格

表格共 15 列：

| 列名 | 说明 | 数据来源 |
|------|------|---------|
| 序号 | 自动递增编号 | 自动生成 |
| 原文件名 | 源文件名 | 自动读取 |
| 新文件名 | 重命名后的文件名 | 自动组合生成 |
| 正则结果 | 从文件名提取的关键字 | 正则匹配 |
| 订单号 | 订单编号 | Excel匹配 / 手动输入 |
| 材料 | 材料类型 | Excel匹配 / 材料对话框 |
| 数量 | 印刷数量 | Excel匹配 / 材料对话框 |
| 尺寸 | 成品尺寸 | PDF自动读取 / 手动输入 |
| 形状 | 形状代号 | 材料对话框 |
| 列组合 | Excel多列组合值 | Excel匹配 |
| 平张 | 平张排版布局数 | 自动计算 |
| 卷装 | 卷装排版布局数 | 自动计算 |
| 工艺 | 颜色模式+膜类型 | 材料对话框 |
| 页数 | PDF页数 | 自动读取 |
| 时间 | 处理时间 | 自动记录 |

### 右键菜单

| 右键位置 | 可用操作 |
|---------|---------|
| 材料列 | 快捷选择材料 |
| 数量列 | 批量设置/增减数量 |
| 尺寸列 | 设置出血值 |
| 订单号列 | 批量生成订单号 |
| 原文件名列 | 提取数量 |
| 任意列 | 复制、删除行、刷新 |

> 右键列头可以隐藏不需要的列、保存列配置或恢复原始布局。

## 新文件名生成规则

新文件名由多个字段按特定顺序组合，组合规则由 **事件分组配置**（设置 → 事件分组）决定。

默认格式：`正则结果_订单号_材料_数量_尺寸_形状_工艺.pdf`

示例：`1234_ORD001_PET_500_210×297+3_R_彩色光膜.pdf`

状态栏右侧的 **"组合"** 预览实时显示当前配置效果。
