# Excel 数据导入与匹配

> 来源：`docs/用户操作教程.md` §4

Excel 数据导入是批量模式的核心支撑，通过 Excel 表格中的数据自动填充文件参数。

## 导入 Excel 数据

1. 点击 **"导入表格"** 按钮
2. 点击 **"选择文件"**，选择 Excel 文件（支持 `.xlsx`、`.xls`）
3. 配置参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| **搜索列** | Excel 中用于匹配的列 | 选择"订单号"列 |
| **返回列** | 匹配成功后提取数据的列 | 选择"数量"列 |
| **序号列** | 可选，提取序号的列 | 选择"序号"列 |
| **正则表达式** | 从文件名提取匹配关键字 | `(\d{4})` |
| **列组合** | 可选，多列用分隔符组合 | 选择"客户+备注"列 |

4. 点击 **"确定"** 完成导入

## 数据匹配流程

```
文件名 → 正则提取关键字 → 在Excel搜索列中查找 → 提取返回列/序号列数据 → 填入表格
```

**匹配规则**：
- **精确匹配**：提取的关键字必须与 Excel 搜索列的值完全一致
- **一对多匹配**：一个关键字匹配多行 Excel 数据时，为每行创建文件副本
- 匹配成功的行背景变为**浅绿色**

## 手动触发匹配

修改正则规则后，点击 **"应用数据"** 按钮重新执行匹配。

## Excel 格式建议

| 要求 | 说明 |
|------|------|
| 首行为表头 | 系统自动跳过首行 |
| 搜索列无重复值 | 如有重复，取第一个匹配项 |
| 数据格式统一 | 订单号等字段格式保持一致 |
| 无合并单元格 | 合并单元格可能导致读取异常 |

**示例 Excel 表格**：

| 订单号 | 客户名 | 数量 | 材料 | 备注 |
|--------|--------|------|------|------|
| 1001 | 张三 | 500 | PET | 加急 |
| 1002 | 李四 | 1000 | PP | 常规 |
| 1003 | 王五 | 2000 | PVC | 特殊工艺 |
