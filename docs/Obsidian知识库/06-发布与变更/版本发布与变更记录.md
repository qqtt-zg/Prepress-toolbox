# 版本发布与变更记录

## 用途
本页是发布、安装包构建、版本说明与历史变更资料的统一入口。

## 当前有效入口
- `docs/发布安装包流程_Skill.md` — 安装包发布流程与操作 Skill
- `installers/Setup.iss` — Inno Setup 安装脚本
- `src/WindowsFormsApp3/Properties/AssemblyInfo.cs` — 程序版本信息

## 发布流程
1. 更新版本号
2. 构建 Release
3. 使用 Inno Setup 生成安装包
4. 验证安装包启动、配置目录、外部工具路径
5. 记录版本说明
