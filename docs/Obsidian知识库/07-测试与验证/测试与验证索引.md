# 测试与验证索引

## 用途
本页是功能验证、专项测试、安装包测试和历史测试指南的统一入口。

## 通用验证流程
1. 确认测试目标属于哪个模块
2. 阅读对应模块的当前有效说明
3. 执行手工流程或自动化测试
4. 记录异常、输入文件、输出文件和复现步骤
5. 将阶段性测试记录归档到 [[归档总索引]]
