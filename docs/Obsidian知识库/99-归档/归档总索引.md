# 归档总索引

## 用途
本页是 `99-归档/` 的总入口，用于查找历史总结、阶段记录、修复诊断、旧测试指南和版本发布记录。

## 归档规则
- 归档文档可以作为历史背景引用。
- 归档文档不作为当前有效说明。
- 当前行为以模块页和“当前有效说明”为准。
- 新文档如果是阶段性总结、修复记录、旧测试指南，应优先进入本目录下对应分类。
