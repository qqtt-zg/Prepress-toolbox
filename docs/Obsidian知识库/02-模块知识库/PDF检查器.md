# PDF检查器

## 当前状态
已实现 PDF 预览、检查、字体分析、页面框信息查看和部分后处理承载能力。

## 模块定位
用于 PDF 预览、检查、字体分析、页面框信息查看，以及部分后处理能力。当前应理解为“检查优先、处理承载为辅”的 PDF 诊断入口。

## 当前主线结论
PDF 检查器负责加载 PDF、展示检查信息，并在需要时触发后处理动作（如字体转曲）；处理完成后，由用户手动决定保存或另存为。
