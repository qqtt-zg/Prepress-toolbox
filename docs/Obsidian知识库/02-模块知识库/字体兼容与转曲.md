# 字体兼容与转曲

## 当前状态
已实现字体兼容处理与 Ghostscript 转曲工作流，当前有效行为是“内存返回 + 用户手动保存”。

## 模块定位
该主题覆盖：
- 系统字体兼容性
- 中文字体筛选
- 字体格式支持
- PDF 字体检测
- Ghostscript 转曲流程

## 当前有效入口
- [[字体转曲当前有效说明]]
- [[PDF检查器当前有效说明]]

## 当前主线结论
`字体兼容与转曲` 当前应理解为一条字体兼容修复工作流，而不是“点击一次就自动导出完成文件”的按钮功能。当前有效行为是：处理完成后返回结果，由上层界面承载，并由用户手动保存。
