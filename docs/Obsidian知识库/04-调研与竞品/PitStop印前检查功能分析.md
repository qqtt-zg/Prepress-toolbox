# PitStop Pro 印前检查功能分析

## 竞品定位
**Enfocus PitStop Pro** 是印刷行业标准的 PDF 编辑与预检工具，全球超过 13 万家印刷厂使用，是 Acrobat 插件生态中最核心的印前检查产品。

- 价格：PitStop Pro ~€599/许可；PitStop Server 需报价
- 目标市场：印前服务、出版、包装、标签印刷
- 核心价值：在印刷前检测和修复 PDF 问题，避免返工成本

## 与本项目的关系
PitStop Pro 是 [[PDF检查器]] 和 [[字体兼容与转曲]] 的核心规格参考来源。其 Inspector 和 Preflight 定义了"PDF 检查工具应具备什么"。

## 核心功能模块

### 1. 印前检查（Preflight） — 复刻价值：极高
自动检测 PDF 中影响印刷质量的问题，生成详细报告。

| 检查类别 | 检查项 | 状态 |
|---------|--------|------|
| 字体问题 | 缺失/损坏字体、未嵌入字体、子集嵌入 | ✅ 已实现 |
| 图像问题 | 低分辨率（<300 DPI）、RGB 应为 CMYK | ⏳ 部分 |
| 页面框问题 | 缺少出血边、尺寸不正确、关系错误 | ✅ 已实现 |
| 颜色问题 | RGB 颜色、专色使用不当、叠印异常 | ⏳ 待实现 |
| 标准符合性 | PDF/X-1a、PDF/X-3、PDF/X-4 验证 | ⏳ 待实现 |

### 2. Inspector（检查器） — 复刻价值：极高
深入剖析 PDF 底层结构，查看任意对象详细参数。

| 检查内容 | 状态 |
|---------|------|
| 页面框参数（MediaBox / CropBox / TrimBox / BleedBox / ArtBox） | ✅ 已实现 |
| 字体信息（名称、类型、嵌入状态、使用页面） | ✅ 已实现 |
| 对象信息（文本/图像/矢量的详细参数） | ⏳ 部分 |
| 颜色信息（色彩空间、颜色值、专色、叠印） | ⏳ 待实现 |

### 3. 字体转曲（Convert to Outlines） — 复刻价值：极高
- 将文字转换为矢量路径，不再依赖字体文件
- ✅ 已使用 Ghostscript `-dNoOutputFonts` 实现
- 当前工作流：内存转曲 + 用户手动保存（参见 [[字体转曲当前有效说明]]）

### 4. 出血检测与添加 — 复刻价值：极高
- 检测功能 ✅ 已实现（检查四边出血值、验证标准）
- 添加功能 ⏳ 待实现（自动扩展内容到出血区域）

### 5. PDF 深度编辑 — 复刻价值：高
- 文本编辑、图像编辑、矢量图形编辑
- 颜色管理（RGB ↔ CMYK、ICC Profile）
- 图层管理、页面操作
- 复刻难度：困难（需要深度 PDF 内容流操作）

### 6. 裁切标记和套准标记 — 复刻价值：高
- 裁切标记、套准标记、出血标记、颜色条
- 复刻难度：简单到中等

### 7. 批量处理与自动化（Action List） — 复刻价值：极高
- 录制操作序列、保存为 Action List、批量应用
- 条件执行、错误处理
- 详见 [[ActionList架构方向]]

## 已有优势
- 完整的页面框检查
- 准确的字体检测（Poppler）
- 稳定的字体转曲（Ghostscript）
- 现代化 UI 设计

## 主要差距
- 缺少裁切和套准标记
- 无法自动添加出血
- 颜色转换功能缺失
- 图像编辑能力有限
- 批量处理不完善

## 对本项目的指导意义

### 与 Quite Imposing 的协同
- **PitStop** → 检查与修复（PDF 检查器方向）
- **Quite Imposing** → 拼版与输出（排版工作区方向）
- 两者互补，合在一起构成完整印前解决方案

### 第一阶段（核心印前）
1. 裁切标记添加
2. 套准标记添加
3. 出血自动添加
4. 图像 DPI 检测

### 第二阶段（增强功能）
1. RGB → CMYK 转换
2. 批量处理完善
3. 预检配置
4. 对象编辑
