# 术语表

## 导航与界面
- **首页**：文件批量重命名的主面板，对应 `FileRenamePanel`
- **数据库**：Excel 导入结果的查看面板，对应 `DatabasePanel`，不是独立数据库系统
- **工作台**：当前项目里存在歧义，需结合 [[工作台与排版工作区]] 理解
- **排版工作区**：偏专业拼版与参数设置的工作区，对应 `ImpositionWorkspacePanel`
- **AE 风格工作台**：偏多 PDF 管理、预览、日志的工作台，对应 `AeWorkspacePanel`

## 印前与拼版
- **骑马订**：书册页序嵌套排列
- **N-Up**：多页合一拼版
- **Step & Repeat / 连拼**：同一内容重复排列
- **Shuffle / 页面重排**：按指定顺序重排页面
- **Cut & Stack**：裁切后按堆叠顺序输出
- **Bleed / 出血**：裁切线外延伸区域
- **Crop Marks / 裁切标记**：裁切位置标记
- **Registration Marks / 套准标记**：多色对位标记

## 文档状态标签
- **当前有效**：可作为当前实现与使用依据
- **历史记录**：描述已发生过的设计或修复过程
- **待整理**：已识别重要，但尚未融入主知识页
- **已过期**：信息与当前实现明显不一致，仅保留历史参考

## 阅读建议
遇到同一概念多种叫法时，优先回到模块页和 `代码入口与模块映射` 交叉确认。
