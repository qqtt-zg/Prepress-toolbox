# 项目总览

## 项目定位
`Prepress-toolbox` 是面向印刷行业印前处理的 Windows Forms 桌面工具，当前已形成以下主能力：
- 文件批量重命名
- Excel 数据导入与匹配
- PDF 检查、预览与字体处理
- PDF 拆分
- 拼版与排版相关能力
- 综合设置与主题配置

## 当前最需要收口的问题
1. 工作台、排版工作区、AE 风格工作台定义不一致
2. PDF 检查器与字体转曲文档过多，当前有效说明不集中
3. 数据库模块命名容易误导，实际更接近 Excel 结果视图
4. PRD、用户手册、调研、实现总结之间缺少统一主链
