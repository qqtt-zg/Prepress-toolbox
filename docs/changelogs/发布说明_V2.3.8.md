## 版本更新内容

### 主要更新：PDF操作界面进度反馈优化

#### 1. 进度条叠加层实现 ⭐
- **StatusBarProgressOverlay 控件**：全新的进度叠加层控件
- **视觉效果**：4px 高度的半透明进度条，叠加在状态栏上方
- **文字固定**：状态栏文字信息位置保持不变，不再被进度条推动
- **双模式支持**：
  - 确定进度模式：显示 0-100% 的精确进度
  - 不确定进度模式：显示流畅的滚动动画

#### 2. 位置精确控制
- **从最左端开始**：进度条从 x=0 开始，覆盖整个状态栏宽度
- **不受 padding 影响**：使用 Anchor 布局，忽略父容器的内边距
- **动态宽度调整**：窗口大小改变时自动调整进度条宽度

#### 3. 性能优化
- **双缓冲绘制**：无闪烁，流畅显示
- **线程安全**：支持跨线程调用，自动切换到 UI 线程
- **渐变效果**：使用 LinearGradientBrush 和高光效果，视觉更优雅

#### 4. 用户体验提升
- **清晰的进度反馈**：操作进度一目了然
- **专业的视觉效果**：半透明叠加，不遮挡信息
- **流畅的动画**：30ms 刷新间隔，动画流畅不卡顿
