## 技术细节

### 进度条配置
- **高度**: 4px
- **位置**: Dock.Top（在状态栏内部顶部）
- **颜色**: `Color.FromArgb(180, 24, 144, 255)` - 半透明粉紫色
- **背景**: `Color.FromArgb(50, 255, 255, 255)` - 非常淡的白色

### 使用场景

#### 1. 不确定进度（滚动动画）
用于无法预知进度的操作：
- PDF 文件加载
- 初始化浏览器
- 字体转曲开始

```csharp
ShowBottomProgress(); // 自动启动滚动动画
```

#### 2. 确定进度（百分比）
用于可以跟踪进度的操作：
- 文件保存（0% → 30% → 70% → 100%）
- 批量处理
- 添加标记

```csharp
ShowBottomProgress();
UpdateBottomProgress(30);
UpdateBottomProgress(70);
UpdateBottomProgress(100);
HideBottomProgress();
```

## 实际应用示例

### 文件保存
```csharp
private async void BtnSave_Click(object sender, EventArgs e)
{
    ShowBottomProgress();
    UpdateBottomProgress(30);
    
    await Task.Run(() => File.WriteAllBytes(_originalFilePath, _currentPdfBytes));
    
    UpdateBottomProgress(70);
    // ... 更多操作
    UpdateBottomProgress(100);
    
    await Task.Delay(100);
    HideBottomProgress();
}
```

### PDF 加载
```csharp
private async Task LoadPdfFileAsync(string filePath)
{
    ShowBottomProgress();
    UpdateBottomProgress(10);
    
    _originalPdfBytes = File.ReadAllBytes(filePath);
    UpdateBottomProgress(40);
    
    await _pdfPreview.LoadPdfAsync(filePath);
    UpdateBottomProgress(100);
    
    await Task.Delay(100);
    HideBottomProgress();
}
```

### 字体转曲
```csharp
private void Inspector_FontOutlineStarted(object sender, EventArgs e)

{
```
