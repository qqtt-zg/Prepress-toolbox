## 🔧 技术实现

### Ghostscript 集成

**检测到的版本**：10.06.0  
**安装路径**：`C:\Program Files\gs\gs10.06.0\bin\gswin64c.exe`

**命令行参数**：
```bash
gswin64c.exe -o "output.pdf" \
  -dNoOutputFonts \
  -sDEVICE=pdfwrite \
  -dPDFSETTINGS=/prepress \
  -dCompatibilityLevel=1.4 \
  "input.pdf"
```

**自动检测逻辑**：
1. 检查应用程序目录
