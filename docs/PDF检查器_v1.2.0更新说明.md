# PDF检查器 v1.2.0 更新说明

## 🎉 重大更新：独立窗口模式

**发布日期**: 2026-01-19  
**版本**: v1.2.0  
**类型**: 重大功能更新

---

## 📋 更新概述

PDF检查器现在采用**独立窗口模式**，完全模仿**Enfocus PitStop Pro**的Inspector窗口行为，提供更专业、更灵活的用户体验。

---

## ✨ 新增功能

### 1. 独立浮动窗口 ⭐

检查器现在是一个独立的浮动窗口，而不是侧边栏。

**特性**：
- ✅ 独立的窗口，可自由移动
- ✅ 可调整大小
- ✅ 不在任务栏显示
- ✅ 关闭时隐藏而不是销毁
- ✅ 始终在主窗口之上

**优势**：
- 更符合专业软件习惯
- 不占用主窗口空间
- 多显示器友好
- 更灵活的布局

### 2. 智能窗口定位 ⭐

窗口会智能地定位到合适的位置。

**首次打开**：
- 自动定位到主窗口右侧
- 保持10像素间距
- 确保不超出屏幕范围

**再次打开**：
- 恢复上次的位置
- 恢复上次的尺寸
- 自动检测多显示器

**智能检测**：
- 检测窗口是否在可见区域
- 如果不可见，自动重新定位
- 支持多显示器配置

### 3. 窗口位置记忆 ⭐

窗口位置和尺寸会自动保存。

**自动保存**：
- 移动窗口 → 自动保存位置
- 调整大小 → 自动保存尺寸
- 实时保存，无需手动操作

**自动恢复**：
- 下次打开 → 恢复到上次位置
- 下次打开 → 恢复到上次尺寸
- 跨会话保持

### 4. 双向同步增强 ⭐

预览和检查器之间的同步更加智能。

**预览 → 检查器**：
- 切换页面 → 检查器自动更新
- 打开新PDF → 检查器自动加载
- 实时同步，无延迟

**检查器 → 预览**：
- 点击页面 → 预览自动跳转
- 点击问题 → 预览跳转到问题页
- 双向同步，无缝体验

## 🚀 使用方法

### 基本操作

#### 打开检查器
```
1. 打开PDF文件
2. 点击工具栏的"检查器"按钮
3. 检查器窗口在主窗口右侧弹出
```

#### 移动窗口
```
1. 拖动窗口标题栏
2. 移动到任意位置
3. 位置自动保存
```

#### 调整大小
```
1. 拖动窗口边缘
2. 调整到合适大小
3. 尺寸自动保存
```

#### 多显示器使用
```
1. 拖动窗口到第二个显示器
2. 调整到合适位置
3. 下次打开自动恢复到第二个显示器
```

#### 关闭窗口
```
1. 点击窗口关闭按钮
2. 窗口隐藏（不销毁）
3. 再次点击"检查器"按钮重新显示
```

## 🎯 技术细节

### 核心实现

#### 独立窗口类
```csharp
public class PdfInspectorForm : Form
{
    // 特性
    ShowInTaskbar = false;  // 不在任务栏显示
    Owner = mainForm;       // 设置主窗口为所有者
    
    // 关闭时隐藏
    FormClosing += (s, e) => {
        if (e.CloseReason == CloseReason.UserClosing) {
            e.Cancel = true;
            this.Hide();
        }
    };
}
```

#### 位置记忆
```csharp
// 保存
AppSettings.Set("PdfInspector_WindowX", this.Location.X);
AppSettings.Set("PdfInspector_WindowY", this.Location.Y);

// 恢复
var x = AppSettings.Get("PdfInspector_WindowX");
var y = AppSettings.Get("PdfInspector_WindowY");
this.Location = new Point(x, y);
```

#### 智能定位
```csharp
// 主窗口右侧
int x = mainForm.Right + 10;
int y = mainForm.Top;

// 确保不超出屏幕
if (x + this.Width > screen.WorkingArea.Right) {
    x = screen.WorkingArea.Right - this.Width - 10;
}
```

## ⚠️ 破坏性变更

### 移除的功能

1. **侧边栏模式** - 已完全移除
   - 不再使用SplitContainer
   - 不再有Panel2
   - 不再有折叠/展开功能

2. **内嵌检查器** - 已移除
   - PdfInspectorControl不再直接添加到主面板
   - 改为添加到独立窗口

### 迁移指南

如果你之前使用了侧边栏模式的代码：

**旧代码（v1.1）**：
```csharp
// 切换侧边栏
panel.ToggleInspector();

// 检查是否可见
if (panel.InspectorVisible) { ... }
```

**新代码（v1.2）**：
```csharp
// 显示独立窗口
panel.ShowInspector();

// 检查窗口是否可见
if (inspectorForm != null && inspectorForm.Visible) { ... }
```
