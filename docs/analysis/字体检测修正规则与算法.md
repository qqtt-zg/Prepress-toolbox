# 字体检测修正规则与算法

## 核心修正算法

### 1. 列位置固定解析算法

#### 原理

pdffonts工具输出使用**固定列宽格式**，而非简单的空格分隔。这是类似于`ls -l`命令的格式化输出。

#### 列宽定义

```
列名         起始位置  宽度  结束位置  说明
─────────────────────────────────────────────────────
name         0         37    36        字体名称（可能包含子集前缀）
type         37        18    54        字体类型（可能包含空格）
encoding     55        17    71        字符编码
emb          72        4     75        是否嵌入（yes/no）
sub          76        4     79        是否子集（yes/no）
uni          80        4     83        是否Unicode（yes/no）
object ID    84        -     -         PDF对象ID
```

#### 实现代码

```csharp
private FontInfo ParseFontLine(string line)
{
    // 1. 验证行长度
    if (line.Length < 75)
    {
        return null; // 行太短，无法包含必需的emb列
    }

    // 2. 按固定位置提取字段
    string name     = ExtractColumn(line, 0, 37);
    string type     = ExtractColumn(line, 37, 18);
    string encoding = ExtractColumn(line, 55, 17);
    string emb      = ExtractColumn(line, 72, 4);
    string sub      = ExtractColumn(line, 76, 4);
    string uni      = ExtractColumn(line, 80, 4);

    // 3. 创建FontInfo对象
    var fontInfo = new FontInfo
    {
        FontName = name,
        FontSubtype = type,
        Encoding = encoding
    };

    // 4. 判断嵌入状态
    fontInfo.EmbeddingStatus = DetermineEmbeddingStatus(emb, sub);

    // 5. 移除子集前缀
    fontInfo.FontName = RemoveSubsetPrefix(fontInfo.FontName);

    return fontInfo;
}

private string ExtractColumn(string line, int start, int width)
{
    if (line.Length <= start)
        return "";
    
    int actualWidth = Math.Min(width, line.Length - start);
    return line.Substring(start, actualWidth).Trim();
}
```

### 2. 嵌入状态判断算法

#### 决策树

```
emb列的值
├─ "yes" (嵌入)
│  ├─ sub="yes" → SubsetEmbedded (子集嵌入)
│  └─ sub="no"  → FullyEmbedded (完全嵌入)
└─ "no" (未嵌入)
   └─ → NotEmbedded (未嵌入)
```

#### 实现代码

```csharp
private FontEmbeddingStatus DetermineEmbeddingStatus(string emb, string sub)
{
    bool isEmbedded = emb.Equals("yes", StringComparison.OrdinalIgnoreCase);
    bool isSubset = sub.Equals("yes", StringComparison.OrdinalIgnoreCase);

    if (isEmbedded)
    {
        return isSubset ? FontEmbeddingStatus.SubsetEmbedded 
                        : FontEmbeddingStatus.FullyEmbedded;
    }
    else
    {
        return FontEmbeddingStatus.NotEmbedded;
    }
}
```

#### 特殊情况处理

| emb | sub | 说明 | 处理方式 |
|-----|-----|------|---------|
| yes | yes | 正常：子集嵌入 | SubsetEmbedded |
| yes | no | 正常：完全嵌入 | FullyEmbedded |
| no | no | 正常：未嵌入 | NotEmbedded |
| no | yes | **异常**：理论上不可能 | 视为NotEmbedded |
| "" | "" | 解析失败 | 默认NotEmbedded + 记录警告 |

### 3. 子集前缀移除算法

#### 规则

PDF规范定义子集字体名称格式：
- 前缀：6个大写字母（A-Z）
- 分隔符：加号（+）
- 实际字体名：任意字符

**示例**：
- `EPROCH+MicrosoftYaHei` → `MicrosoftYaHei`
- `ABCDEF+Arial-Bold` → `Arial-Bold`
- `Times-Roman` → `Times-Roman`（无前缀，保持不变）

#### 实现代码

```csharp
private string RemoveSubsetPrefix(string fontName)
{
    // 正则表达式: ^[A-Z]{6}\+
    // ^ : 字符串开头
    // [A-Z]{6} : 恰好6个大写字母
    // \+ : 加号（需要转义）
    
    if (Regex.IsMatch(fontName, @"^[A-Z]{6}\+"))
    {
        return fontName.Substring(7); // 跳过前7个字符（6字母+1加号）
    }
    
    return fontName;
}
```

#### 边界情况

| 输入 | 输出 | 说明 |
|------|------|------|
| `ABCDEF+Arial` | `Arial` | 标准子集前缀 |
| `ABC+Arial` | `ABC+Arial` | 前缀不足6字母，保持不变 |
| `ABCDEFG+Arial` | `ABCDEFG+Arial` | 前缀超过6字母，保持不变 |
| `abcdef+Arial` | `abcdef+Arial` | 小写字母，保持不变 |
| `+Arial` | `+Arial` | 无前缀字母，保持不变 |
| `ABCDEF-Arial` | `ABCDEF-Arial` | 非加号分隔，保持不变 |

### 4. 字体类型映射算法

#### 映射表

| Poppler输出 | 内部表示 | Acrobat显示 | 说明 |
|------------|---------|------------|------|
| `Type 1` | Type1 | Type 1 | PostScript Type 1字体 |
| `Type 1C` | Type1C | Type 1 (CFF) | 压缩的Type 1字体 |
| `Type 3` | Type3 | Type 3 | 用户定义字体（位图） |
| `TrueType` | TrueType | TrueType | TrueType字体 |
| `CID Type 0` | CIDType0 | CIDFont Type 0 | CID PostScript字体 |
| `CID Type 0C` | CIDType0C | CIDFont Type 0 (CFF) | CID PostScript CFF字体 |
| `CID TrueType` | CIDTrueType | CIDFont Type 2 | CID TrueType字体 |

```csharp
public enum FontType

private FontType ParseFontType(string typeString)
{
    switch (typeString.Trim())

        case "Type 1":
            return FontType.Type1;
        case "Type 1C":
            return FontType.Type1C;
        case "Type 3":
            return FontType.Type3;

            return "Type 3";

        case FontType.CIDType0:
            return "CIDFont Type 0";
        case FontType.CIDType0C:
            return "CIDFont Type 0 (CFF)";
        case FontType.CIDTrueType:
            return "CIDFont Type 2";
        default:
            return "Unknown";
    }
}
```

## 差异修正规则

### 规则1: 字体名称标准化

**问题**：不同工具显示的字体名称可能有细微差异

**修正规则**：
1. 移除子集前缀（`^[A-Z]{6}\+`）
2. 去除首尾空格
3. 统一大小写（可选，根据需求）

**示例**：
```
Poppler:  "EPROCH+MicrosoftYaHei  "
修正后:   "MicrosoftYaHei"

Poppler:  "Times-Roman"
修正后:   "Times-Roman"
```

### 规则2: 嵌入状态标准化

**问题**：不同工具使用不同的术语

**映射表**：

| Poppler | Acrobat | 标准状态 |
|---------|---------|---------|
| emb=yes, sub=yes | 嵌入子集 | SubsetEmbedded |
| emb=yes, sub=no | 嵌入 | FullyEmbedded |
| emb=no, sub=no | 未嵌入 | NotEmbedded |

**显示文本映射**：
```csharp

            return "完全嵌入";
        case FontEmbeddingStatus.NotEmbedded:
            return "未嵌入";
        default:
            return "未知";
    }
}
```

### 规则3: 字体类型标准化

**问题**：Poppler和Acrobat使用不同的类型名称

**修正规则**：
- 保留Poppler原始类型用于内部处理
- 提供Acrobat兼容的显示名称
- 支持双向映射

**实现**：
```csharp
public class FontInfo
{
    // 内部存储Poppler原始类型
    public string FontSubtype { get; set; }
    
    // 提供Acrobat兼容的显示名称
    public string DisplayType => GetAcrobatDisplayName(FontSubtype);
    
    private string GetAcrobatDisplayName(string popplerType)
    {
        var mapping = new Dictionary<string, string>
        {
            { "Type 1", "Type 1" },
            { "Type 1C", "Type 1 (CFF)" },
            { "Type 3", "Type 3" },
            { "TrueType", "TrueType" },
            { "CID Type 0", "CIDFont Type 0" },
            { "CID Type 0C", "CIDFont Type 0 (CFF)" },
            { "CID TrueType", "CIDFont Type 2" }
        };
        
        return mapping.TryGetValue(popplerType, out var displayName) 
            ? displayName 
            : popplerType;
    }
}
```

### 规则4: 标准字体识别

**问题**：标准字体即使未嵌入也不应标记为问题

**标准字体列表**（PDF 1.7规范）：
```csharp
private static readonly HashSet<string> StandardFonts = new HashSet<string>(
```

## 验证算法

### 1. 一致性验证

```csharp
public class ConsistencyValidator
{
    public ValidationResult Validate(FontInfo popplerFont, AcrobatFontInfo acrobatFont)
    {
        var result = new ValidationResult();
        
        // 1. 验证字体名称
        result.NameMatch = CompareFontNames(
            popplerFont.FontName, 
            acrobatFont.Name
        );
        
        // 2. 验证嵌入状态
        result.EmbeddingMatch = CompareEmbeddingStatus(
            popplerFont.EmbeddingStatus,
            acrobatFont.EmbeddingStatus
        );
        
        // 3. 验证字体类型
        result.TypeMatch = CompareFontTypes(
            popplerFont.FontSubtype,
            acrobatFont.Type
        );
        
        return result;
    }
    
    private bool CompareFontNames(string poppler, string acrobat)
    {
        // 移除子集前缀后比较
        string cleanPoppler = RemoveSubsetPrefix(poppler);
        string cleanAcrobat = RemoveSubsetPrefix(acrobat);
        
        return string.Equals(cleanPoppler, cleanAcrobat, 
            StringComparison.OrdinalIgnoreCase);
    }
    
    private bool CompareEmbeddingStatus(
        FontEmbeddingStatus popplerStatus, 
        string acrobatStatus)
    {
        var mapping = new Dictionary<FontEmbeddingStatus, string[]>
        {
            { FontEmbeddingStatus.SubsetEmbedded, 
              new[] { "嵌入子集", "Embedded Subset", "Subset" } },
            { FontEmbeddingStatus.FullyEmbedded, 
              new[] { "嵌入", "Embedded", "Full" } },
            { FontEmbeddingStatus.NotEmbedded, 
              new[] { "未嵌入", "Not Embedded", "None" } }
        };
        
        if (mapping.TryGetValue(popplerStatus, out var validTexts))
        {
            return validTexts.Any(t => 
                acrobatStatus.Contains(t, StringComparison.OrdinalIgnoreCase));
        }
        
        return false;
    }
}
```

### 2. 准确率计算

```csharp
public class AccuracyCalculator
{
    public AccuracyReport Calculate(
        List<FontInfo> popplerFonts,
        List<AcrobatFontInfo> acrobatFonts)
    {
        int totalFonts = acrobatFonts.Count;
        int correctNames = 0;
        int correctEmbedding = 0;
        int correctTypes = 0;
        
        foreach (var acrobatFont in acrobatFonts)
        {
            var popplerFont = FindMatchingFont(popplerFonts, acrobatFont);
            if (popplerFont == null)
                continue;
            
            if (CompareFontNames(popplerFont.FontName, acrobatFont.Name))
                correctNames++;
            
            if (CompareEmbeddingStatus(popplerFont.EmbeddingStatus, acrobatFont.EmbeddingStatus))
                correctEmbedding++;
            
            if (CompareFontTypes(popplerFont.FontSubtype, acrobatFont.Type))
                correctTypes++;
        }
        
        return new AccuracyReport
        {
            TotalFonts = totalFonts,
            NameAccuracy = (double)correctNames / totalFonts,
            EmbeddingAccuracy = (double)correctEmbedding / totalFonts,
            TypeAccuracy = (double)correctTypes / totalFonts,
            OverallAccuracy = (double)(correctNames + correctEmbedding + correctTypes) / (totalFonts * 3)
        };
    }
}

public class AccuracyReport
{
    public int TotalFonts { get; set; }
    public double NameAccuracy { get; set; }
    public double EmbeddingAccuracy { get; set; }
    public double TypeAccuracy { get; set; }
    public double OverallAccuracy { get; set; }
    
    public override string ToString()
    {
        return $@"
字体检测准确率报告
─────────────────────
总字体数: {TotalFonts}
字体名称准确率: {NameAccuracy:P2}
嵌入状态准确率: {EmbeddingAccuracy:P2}
字体类型准确率: {TypeAccuracy:P2}
总体准确率: {OverallAccuracy:P2}
";
    }
}
```

### 2. 数据验证规则

```csharp
private bool ValidateFontInfo(FontInfo fontInfo)
{
    var errors = new List<string>();
    
    // 1. 字体名称不能为空
    if (string.IsNullOrWhiteSpace(fontInfo.FontName))
    {
        errors.Add("字体名称为空");
    }
    
    // 2. 字体类型必须有效
    if (string.IsNullOrWhiteSpace(fontInfo.FontSubtype))
    {
        errors.Add("字体类型为空");
    }
    
    // 3. 嵌入状态必须有效
    if (!Enum.IsDefined(typeof(FontEmbeddingStatus), fontInfo.EmbeddingStatus))
    {
        errors.Add("嵌入状态无效");
    }
    
    if (errors.Any())
    {
        LogHelper.Warn($"字体信息验证失败: {string.Join(", ", errors)}");
        return false;
    }
    
    return true;
}
```

### 3. 回退机制

```csharp
public DocumentFontInfo InspectFonts(string filePath)
{
    try
    {
        // 1. 尝试使用Poppler
        if (IsPdffontsAvailable())
        {
            var result = InspectWithPoppler(filePath);
            if (result != null && result.Fonts.Any())
            {
                return result;
            }
        }
        
        // 2. 回退到iText
        LogHelper.Warn("Poppler检测失败，回退到iText");
        return InspectWithIText(filePath);
    }
    catch (Exception ex)
    {
        LogHelper.Error($"字体检测失败: {ex.Message}", ex);
        return new DocumentFontInfo { FilePath = filePath };
    }
}
```
