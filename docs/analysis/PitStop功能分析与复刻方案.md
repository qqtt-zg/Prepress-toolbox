## PitStop Pro 简介

**Enfocus PitStop Pro** 是印刷行业标准的 Adobe Acrobat 插件，全球超过 13 万家印刷厂使用。

### 核心价值
- 在印刷前检测和修复 PDF 问题
- 避免返工成本和时间损耗
- 提供专业的印前编辑能力

### 主要应用场景
- 印前检查（Preflight）
- PDF 编辑和修复
- 页面框管理
- 字体和颜色检查
- 图像质量检测

## 核心功能模块分析

### 1. 印前检查（Preflight）⭐⭐⭐⭐⭐

#### 功能描述
自动检测 PDF 文件中影响印刷质量的各类问题，生成详细报告。

#### 检查项目

**字体问题**
- ❌ 缺失或损坏的嵌入字体
- ⚠️ 未嵌入字体
- ⚠️ 子集嵌入字体

**图像问题**
- ❌ 低分辨率图像（<300 DPI）
- ⚠️ 高分辨率图像（>600 DPI，浪费空间）
- ⚠️ RGB 颜色模式（应该是 CMYK）

**页面框问题**
- ❌ 缺少出血边
- ⚠️ 页面框尺寸不正确
- ⚠️ 页面框关系错误

**颜色问题**
- ⚠️ RGB 颜色（印刷应该用 CMYK）
- ⚠️ 专色使用不当
- ⚠️ 透明度和叠印异常

**标准符合性**
- 验证 PDF/X-1a、PDF/X-3、PDF/X-4 等标准
- 确保文件适配印刷生产流程

#### 复刻难度
🟢 **中等** - 可以使用 iText 7 + Poppler 实现大部分检查

#### 复刻价值
⭐⭐⭐⭐⭐ **极高** - 这是印前最核心的功能

---

### 2. PDF 深度编辑 ⭐⭐⭐⭐

#### 功能描述
无需原始文件即可编辑 PDF 中的任意元素。

#### 编辑能力

**文本编辑**
- 修改文字内容
- 更改字体和大小
- 转换为轮廓（字体转曲）
- 移动和调整位置

**图像编辑**
- 重采样（调整分辨率）
- 应用曲线调整
- 调整亮度/对比度
- 压缩和锐化
- 调整尺寸

**矢量图形编辑**
- 移动、缩放、旋转
- 调整填充和描边颜色
- 编辑路径
- 组合和创建新形状

**颜色管理**
- 配置色彩管理设置
- 转换或重映射颜色到不同色彩空间
- RGB ↔ CMYK 转换

**图层管理**
- 创建和选择命名图层
- 将对象分配到图层
- 展平图层结构

**页面操作**
- 调整页面尺寸
- 修改裁切框（Trim Box）
- 添加出血
- 缩放整个文档

#### 复刻难度
🔴 **困难** - 需要深度 PDF 内容流操作

#### 复刻价值
⭐⭐⭐⭐ **高** - 非常实用，但实现复杂

### 3. Inspector（检查器）⭐⭐⭐⭐⭐

深入剖析 PDF 文件的底层结构，查看任意对象的详细参数。

**页面框参数** ✅ 已实现

- CropBox（裁剪框）
- TrimBox（裁切框）
- BleedBox（出血框）
- ArtBox（艺术框）

**对象信息** ⏳ 部分实现
- 文本对象：字体、大小、颜色、位置
- 图像对象：类型、分辨率、色彩空间、尺寸
- 矢量图形：路径、填充、描边、透明度

**字体信息** ✅ 已实现
- 字体名称和类型
- 嵌入状态
- 使用页面
- 字符集和字形

**颜色信息** ⏳ 待实现
- 色彩空间
- 颜色值
- 专色/印刷色
- 叠印设置

#### 复刻难度
🟢 **简单到中等** - 已有基础实现

#### 复刻价值
⭐⭐⭐⭐⭐ **极高** - 印前检查的核心工具

- 鼠标悬停显示信息

#### 编辑功能
- 输入框修改数值
- 拖拽调整边界
- 等比缩放
- 对齐工具
- 批量应用到多页

#### 复刻难度
🟡 **中等** - 需要坐标映射和交互设计

#### 复刻价值
⭐⭐⭐⭐ **高** - 直观易用

### 5. 裁切标记和套准标记 ⭐⭐⭐⭐

#### 功能描述
自动添加印刷所需的各种标记。

#### 标记类型

**裁切标记（Crop Marks）**
- 指示最终裁切位置
- 支持 InDesign 和 QuarkXPress 样式
- 基于 TrimBox 定位

**出血标记（Bleed Marks）**
- 指示出血区域边界
- 基于 BleedBox 定位

**套准标记（Registration Marks）**
- 多色印刷对齐标记
- 十字线或圆形标记
- 放置在页面四角

**颜色条（Color Bars）**
- CMYK 色块
- 用于印刷质量控制

#### 复刻难度
🟢 **简单到中等** - 主要是矢量图形绘制

#### 复刻价值
⭐⭐⭐⭐ **高** - 印刷必需功能

### 7. 字体转曲（Convert to Outlines）⭐⭐⭐⭐⭐

#### 功能描述
将文字转换为矢量路径，不再依赖字体文件。

#### 功能特点 ✅ 已实现
- 保持原始视觉效果
- 不依赖字体文件
- 适合印刷和交付
- 防止字体缺失问题

#### 实现方式
- 使用 Ghostscript `-dNoOutputFonts` 参数
- 高质量输出（`/prepress` 设置）
- PDF 1.4 兼容性

#### 复刻难度
🟢 **简单** - 已使用 Ghostscript 实现

#### 复刻价值
⭐⭐⭐⭐⭐ **极高** - 印刷必备功能

## 技术实现方案

### 核心技术栈

#### PDF 处理库

**iText 7** ✅ 已使用
- 用途：PDF 读取、页面框解析、基础编辑
- 许可证：AGPL（商业使用需购买许可）
- 优点：功能强大、文档完善
- 缺点：商业许可费用高

**Poppler** ✅ 已使用
- 用途：字体检测（pdffonts）
- 许可证：GPL（作为外部工具调用不受限）
- 优点：准确性高、与 Adobe 一致
- 缺点：需要外部可执行文件

**Ghostscript** ✅ 已使用
- 用途：字体转曲、PDF 处理
- 许可证：AGPL（商业使用需购买许可）
- 优点：功能强大、稳定可靠
- 缺点：商业许可费用

**PdfiumViewer** ✅ 已使用
- 用途：PDF 预览渲染
- 许可证：Apache 2.0（可商用）
- 优点：轻量级、渲染快
- 缺点：功能有限

#### 可选技术方案

**PDFsharp** 🔵 可考虑
- 用途：PDF 创建和编辑
- 许可证：MIT（可商用）
- 优点：开源免费、易用
- 缺点：功能相对有限

**ImageMagick** 🔵 可考虑
- 用途：图像处理、格式转换
- 许可证：Apache 2.0
- 优点：功能强大
- 缺点：体积较大

### 架构设计

```
UI 层（WinForms + AntdUI）
    ↓
Presenter 层（MVP 模式）
    ↓
Service 层（业务逻辑）
    ├── PdfInspectorService          # 页面框检查
    ├── PdfFontInspectorService      # 字体检查
    ├── PdfFontOutlineService        # 字体转曲
    ├── PdfImageInspectorService     # 图像检查（待实现）
    ├── PdfColorInspectorService     # 颜色检查（待实现）
    ├── PdfMarkService               # 标记添加（待实现）
    ├── PdfBleedService              # 出血处理（待实现）
    └── PdfBatchProcessingService    # 批量处理（待实现）
    ↓
PDF 库层（iText 7, Poppler, Ghostscript）
```
