## 🔧 技术架构

### 组件层次结构

```
MainShellForm
└── PdfOperationsPanel (PDF操作面板)
    ├── SplitContainer (分割容器)
    │   ├── Panel1 (左侧 - PDF预览)
    │   │   ├── Toolbar (工具栏)
    │   │   │   ├── [打开PDF]
    │   │   │   ├── [保存]
    │   │   │   ├── [裁切标记]
    │   │   │   ├── [套准标记]
    │   │   │   └── [检查器] ← 新增
    │   │   └── CefPdfPreviewControl (PDF预览)
    │   └── Panel2 (右侧 - 检查器) ← 新增
    │       └── PdfInspectorControl
    │           ├── 当前页面标签页
    │           ├── 所有页面标签页
    │           └── 问题标签页
    └── StatusBar (状态栏)
```

### 数据流

```
用户操作
    ↓
[打开PDF] → PdfOperationsPanel.LoadPdfFileAsync()
    ↓
CefPdfPreviewControl.LoadPdfAsync()
    ↓
PdfLoaded事件 → 启用检查器按钮
    ↓
[点击检查器] → ToggleInspector()
    ↓
PdfInspectorControl.LoadPdf()
    ↓
PdfInspectorService.InspectPdf()
    ↓
显示页面框信息和问题列表
```

### 同步机制

```
预览 ←→ 检查器 双向同步

预览切换页面 → PageChanged事件 → 检查器更新
检查器选择页面 → PageSelected事件 → 预览跳转
```
