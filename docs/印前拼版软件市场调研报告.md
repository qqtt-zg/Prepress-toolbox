## 六、针对"中国不干胶标签/吊牌印刷"的关键发现

### 最相关的竞品（直接竞争）

1. **火并 LabelImposer** — 最直接竞品。专为大量短版标签设计，支持可变数据混拼、自动对接数码模切、热文件夹自动化。中国公司，原生中文。

2. **PaSharp（锐图）** — 国产专业包装印前软件，30年积累，覆盖标签/包装全流程（陷印+嵌套+VDP+检测）。北京公司，原生中文。

3. **Compose Express TagEngine** — 吊牌专用端到端方案，从数据库到合版到包装报告全自动化。减少90%+生产时间。

4. **Esko Phoenix + Plato** — 全球最先进的AI拼版引擎，Esko中国有本地化支持（400-608-3756）。价格最高但能力最强。

### 市场空白与机会

1. **价格空白**：火并和PaSharp需询价（通常数万元级别），Montax €299-999但功能不够专业化。**缺少一个价格在￥1,000-5,000区间、专为中国标签印刷优化的轻量级拼版工具**。

2. **中文体验空白**：大多数国际工具无中文界面，国产工具界面现代感不足。

3. **自动化的平民化**：热文件夹+订单驱动的自动化拼版在高端工具中才有，中小标签厂仍大量使用CorelDRAW/AI手动拼版。

4. **数码模切对接**：只有火并LabelImposer明确支持自动输出数码模切文件（DXF），这是一个重要差异化点。

5. **可变数据+混拼**：标签印刷的核心需求——不同尺寸、不同数量的SKU混拼优化，目前仅火并和Compose TagEngine专业支持。

### 市场趋势

- **AI驱动排版**：Esko Phoenix代表的AI优化是行业方向
- **云端/SaaS化**：浏览器端工具正在兴起（PDF Press等），但专业市场仍以桌面为主
- **自动化工作流**：热文件夹 → 自动拼版 → 自动输出 → 数码模切，是标签厂的核心效率提升路径
- **国产替代**：中国印刷企业倾向选择国产印前解决方案（PaSharp、火并等）
