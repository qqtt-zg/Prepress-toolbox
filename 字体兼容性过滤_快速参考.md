## 常见问题

### Q: 加载需要多长时间？
**A:** 通常 10-20 秒，取决于系统字体数量。

### Q: 为什么某些字体不在列表中？
**A:** 因为它们不兼容 PDFsharp，已被自动过滤。

### Q: 可以显示所有字体吗？
**A:** 可以点击"重新加载"，但不推荐，因为会包含不兼容的字体。

### Q: 如何查看被过滤的字体？
**A:** 查看日志文件：`%AppData%\Roaming\大诚重命名工具\Logs\log_YYYYMMDD.txt`

## 技术细节

### 验证方法

```csharp
// 使用实际 PDFsharp API 测试
1. 创建 PDF 文档和页面

3. 绘制测试文本

```

### 验证标准
```
兼容: 能成功创建字体、绘制文本、测量宽度
不兼容: 任何步骤失败（NullReferenceException、ArgumentException 等）
```

### 字体解析器
```
SystemFontResolver: 从 C:\Windows\Fonts 加载字体文件
PdfSharpFontInitializer: 初始化 PDFsharp 字体解析器
支持格式: TTF、TTC、OTF
```
